# Փաստաթղթի նկարագրություն 

Համակարգի հիմք է հանդիսանում փաստաթուղթը իր վիճակների բազմությամբ և իր պրոցեսների տրամաբանությամբ։ Այն պարունակում է տարբեր տիպերի դաշտեր։ Որպես փաստաթղթի դաշտեր կարող են հանդես գալ [մուտքագրման դաշտեր](../rekvizit.html), [աղյուսակներ](../grid.html), [նկար-դաշտեր](../image.html), [մեծ տեքստային դաշտեր](../memo.html), [բաժանարար գծեր](../line.html), [կոճակներ](../button.html)։

Փաստաթղթի նկարագրությունը պարունակում է նաև փաստաթղթի տրամաբանությունը նկարագրելու համար նախատեսված ծրագրային կտոր՝ սկրիպտ։ 
Սկրիպտում գրվում են [համակարգային իրադարձությունների մշակիչներ](../scriptstproced.html), ինչպես նաև փաստաթղթին վերաբերող կամայական այլ ֆունկցիոնալություն։ 

Փաստաթղթի նկարագրության մեջ կարող են լինել [իրավասության նկարագրիչներ](../access.html) և [փաստաթղթի թղթապանակների նկարագրիչներ](../FolderInfo.html)։



## Շարահյուսություն

``` as4x 
DOCUMENT {
    Name = sDocName;
    Caption = sDocCaption;
    ECaption = sDocECaption;
    PrintWidth = nDocPrintWidth;
    PrintFormated = nDocPrintFormated;
    DisableDraft = nDocDisableDraft;
    DisableCopy = nDocDisableCopy;
    Type = nPatternSupport;
    Attachments = nAttachments;
    HI2Support = nHI2Support;
    HISSSupport = nHISSSupport;
    MTSupport = nMTSupport;
    FASupport = nFASupport;
    WordTemplate = stmplNameWord;
    ExcelTemplate = stmplNameExcel;
    Version = nVersion;
    Icon = sIconName;
    Accounting = nAccountingCode;
    ContrSupport = bContrSupport;
    AccessFolder = sAccessFolder;
    StoreInTree = nStoreInTree
    CopyAsRepeatable = nCopyMode
    DisableAttachmentSizeCheck = nDisableAttachsizeCheck;

    'Page ...
    'Access ...
    'FolderInfo ...
    'PrintDataGroup ...

    SCRIPT {
        Subs and Functions 
    };
};
```

Փաստաթղթի նկարագրության բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| sDocName | Փաստաթղթի ներքին անուն։ |
| sDocCaption | Փաստաթղթի անվանում։ |
| sDocECaption | Փաստաթղթի անվանումը օտար լեզվով։ |
| nDocPrintWidth | Փաստաթղթից տպվող տեղեկանքի լայնությունը (նիշերի քանակը)։ |
| nDocPrintFormated | Փաստաթղթից տպվող տեղեկանքի ձևաչափվող լինելու հայտանիշը ([տես նաև](../Functions/AsRepViewer/UseFormatting.md))։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nDocDisableDraft | Անջատում է փաստաթղթի սևագիր ստեղծելու հնարավորությանը։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nDocDisableCopy | Անջատում է փաստաթղթի պատճեն ստեղծելու հնարավորությանը։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nPatternSupport | (Միայն ՀԾ-Ձեռնարկություն) Տիպայի գործողությունների օգտագործման հայտանիշ։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nAttachments | Փաստաթղթին [ֆայլեր կցելու](../Functions/AsAttachment.html) հայտանիշ։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nHI2Support | HI2 աղյուսակի օգտագործման հայտանիշ։ Միացված ժամանակ կատարվում է HI2 և HI2REST աղյուսակներից հաշվառումների ավտոմատ ջնջումը, ինչպես նաև հաշվառումների պահպանման ժամանակ սահմանաչափի ստուգումը։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nHISSSupport | (Միայն ՀԾ-Բանկ) ժամկետային պայմանագրերի հաշվառումների  աղյուսակների օգտագործման հայտանիշ։ Միացված ժամանակ փաստաթուղթը կարող է հաշվառումներ ստեղծել այդ աղյուսակներում։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nMTSupport | (Միայն ՀԾ-Ձեռնարկություն) Նյութական արժեքների հաշվառումների  աղյուսակների օգտագործման հայտանիշ։ Միացված ժամանակ փաստաթուղթը կարող է հաշվառումներ ստեղծել այդ աղյուսակներում։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nFASupport | (Միայն ՀԾ-Ձեռնարկություն) Հիմնական միջոցների հաշվառումների  աղյուսակների օգտագործման հայտանիշ։ Միացված ժամանակ փաստաթուղթը կարող է հաշվառումներ ստեղծել այդ աղյուսակներում։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| stmplNameWord | (Հնացած) Սահմանում է փաստաթղթի Word արտահանման տպվող ձևանմուշի անունը։ Ձևանմուշի անվանման երկարությունը չպետք է գերազանցի 20 նիշը։ Փոխարենը հարկավոր է օգտագործել նոր տպելու ձևանմուշները։ |
| stmplNameExcel | Եթե որպես արժեք տրված է `SYSTEMPLATE`, ապա դեպի Excel արտահանումը կատարվում է համակարգի ստանդարտ ձևաչափով։ <br/>(Հնացած) Սահմանում է փաստաթղթի Excel արտահանման տպվող ձևանմուշի անունը։ Ձևանմուշի անվանման երկարությունը չպետք է գերազանցի 20 նիշը։ Փոխարենը հարկավոր է օգտագործել նոր տպելու ձևանմուշները։ |
| nVersion | (Հնացած) Սահմանում է փաստաթղթի նկարագրության տարբերակի համարը։ Որը ամրագրված է հետագա օգտագործման համար։ |
| sIconName | (Հնացած) Համակարգի կողմից աջակցվող պատկերակի ներքին անուն: |
| nAccountingCode | Սահամանում է հաշվառման ներքին համարը, որին և պատկանում է տվյալ փաստաթղթը։ |
| sAccessFolder | (Միայն ՀԾ-Բանկ) Այն թղթապանակի ներքին անունը, որում պահվում են իրավասությունները։ Չլրացնելու դեպքում իրավասությունների փոփոխումը հնարավոր չի լինի հաստատել։ <br/>*Այս հատկությունը օգտագործվում է միայն հին «բանալիով» իրավասությունների նշանակման համար։* <br/>строковое выражение, определяющее идентификатор папки, содержащее заголовки доступов данного документа. При отсутствии данного параметра повторное подтверждение для документа невозможно. |
| nContrSupport | (Միայն ՀԾ-Ձեռնարկություն) Պայմանագրերի հաշվառումների  աղյուսակների օգտագործման հայտանիշ։ Միացված ժամանակ փաստաթուղթը կարող է հաշվառումներ ստեղծել այդ աղյուսակներում։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nStoreInTree | Ծառի-տեղեկատուի մեջ գրանցման հայտանիշ։ 0 կամ 1 արժեք ընդունող։ Լռությամբ արժեքը 0։ |
| nCopyMode | Սահմանում է օգտագործողի կողմից պատճենման ռեժիմը։ 0 արժեքի դեպքում բոլոր դաշտերը պատճենվում են։ 1 արժեքի դեպքում պատճենվում են այն դաշտերի արժեքները, որոնք պարունակում են `N` հատկանիշ։ Լռությամբ արժեքը 0: |
| nDisableAttachsizeCheck | 1 արժեքի դեպքում անջատվում են փաստաթղթին [կցված մեկ ֆայլի](../Functions/AsAttachment.html) առավելագույն չափի ստուգումը (ըստ `Param("MAXATTACH")` պարամետրի), ինչպես նաև բոլոր կցված ֆայլերի առավելագույն չափի ստուգումը (ըստ `Param("MAXTOTALATTACH")` պարամետրի)։ Լռությամբ արժեքը 0։ |
| Subs and Functions | Փաստաթղթի տրամաբանությունը նկարագրելու համար նախատեսված ծրագրային կտոր՝ սկրիպտ։ |

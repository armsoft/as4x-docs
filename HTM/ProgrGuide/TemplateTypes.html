Типы шаблонов печатних форм

Код Описание
0 MS Word (*.doc). Старый тип шаблона печати MS Word. Заполняется медленно, оставлен для обратной совместимости. Для заполнения требуется Microsoft Word.
1 MS Excel (*.xls). Старый тип шаблона печати MS Excel. Заполняется медленно, оставлен для обратной совместимости. Для заполнения требуется Microsoft Excel.
2 MS Word Open XML (*.docx). Новый тип шаблона печати MS Word.
3 MS Excel Open XML (*.xlsм). Новый тип шаблона печати MS Excel.
4 PDF (*.pdf). Шаблон задается файлом формата Open Document Text (odt), но заполненный файл конвертируется .pdf.
5 Open Document Text (*.odt). Тип шаблона печати Open Document Text.
6 Open Document Spreadsheet (*.ods). Тип шаблона печати Open Document Spreadsheet.
7 Hyper Text Markup Language (*.htm).
8 MS Excel Open XML (*.xlsx).
9 Текстовый файл (*.txt).
A PDF (от HTML).

Событие TemplateSubstitutionParameters

См. также  Пример  Применяется к

Происходит при заполнении шаблона печатной формы документа. Результатом выполнения является объект типа Dictionary. Задает параметры генерации шаблона подстановки событием TemplateSubstitution. Если произходит генерация множества шаблонов печати событие TemplateSubstitutionParameters срабатывет единожды, в то время как событием TemplateSubstitution срабатывает для каждого документа. Данное событие удобно для вывода поьзовательского интерфейса задания параметров печати для документа.

 

Синтаксис

Function TemplateSubstitutionParameters() as Dictionary
   statements
End Function

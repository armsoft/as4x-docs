Внутренние типы данных

В системе имеются внутренние типы данных, предопределяющие хранимые виды данных, с характерными областями допустимых значений. Внутренние типы используются в системе для указания типов реквизитов в документах, значений колонок и передаваемых параметров в источниках данных, полей ввода в диалоговых окнах:

AmAcc
Boolean
C ()
Ch ()
Date
DateLong
DateRep
File
Folder ()
FullTree ()
N ()
NP ()
NumPair ()
Path
Summa
SummaP
Time
TimeLong
Tree ()

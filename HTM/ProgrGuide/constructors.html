Функции, возвращающие ссылку на объект

В скриптовых разделах системных описаний могут вызываться функции, возвращающие ссылку либо на существующий объект или же создающие новый экземпляр объекта. Команда Set присваивает объектной переменной ссылку на объект.
В общем случае ссылка на объект создается следующим способом:
   Set ObjectVariable=функция
Когда объектная переменная становится ненужной, освободите ее, присвоив значение Nothing:
   Set ObjectVariable=Nothing

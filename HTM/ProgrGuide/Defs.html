Системные описания

Логика функционирования системы основана на системных описаниях. В системных описаниях предопределяется внешний вид, поведение и жизненный цикл объектов системы, начиная от создания объекта, изменения его состояний, вплоть до его возможного удаления. В системных описаниях находятся обработчики системных событий, реагирующие на сгенерированные при определенных ситуациях события. Опираясь на имеющееся системное описание, скриптограф может в дальнейшем создавать экземпляры данного вида объектов или загружать уже созданные экземпляры. Системное описание является шаблоном для последующих экземпляров объектов данного типа. 

В системе "AS-3X" существуют следующие типы системных описаний:

  • Описание документа
  • Описание источника данных
  • Описание вида просмотра
  • Описание дерева-справочника
  • Описание отчета
  • Описание учета
  • Описание модуля
  • Описание стиля печати
  • Описание описательного диалога
  • Описание оповещения
     
  • Описание задания
  • Описание дерева навигации
  • Описание шаблона TEMPLATE
  • Описание шаблона WORDTEMPLATE
  • Описание шаблона EXCELTEMPLATE
  • Описание документа группы шаблона печати TemplGrp

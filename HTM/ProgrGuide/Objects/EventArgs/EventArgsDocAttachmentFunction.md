# EventArgsDocAttachmentFunction Class

Պարունակում է տվյալներ փաստաթղթի [AttachmentFunction](../../ScriptProcs/AttachmentFunction.md) իրադարձության մշակիչներին փոխանցելու համար։

| Հատկություն | Նկարագրություն |
|:--|:--|
| `Doc` | Վերադարձնում է իրադարձությունը առաջացնող փաստաթուղթը։ |
| `Name` | Վերադարձնում է գործողության ներքին անունը։ |
| `SelectedAttachments` | Վերադարձնում է կցված ֆայլերի էջում նշվածների բազմություն՝ Collection տիպի։ Տարրերն [AsAttachment](../../Functions/AsAttachment.md) տիպի են։ |

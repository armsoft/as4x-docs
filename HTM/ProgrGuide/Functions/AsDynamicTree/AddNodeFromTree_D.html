Метод AddNodeFromTree

См. также  Пример  Применяется к

Добавляет в динамическое дерево узел с уже существующим TreeId.

Свойство для чтения.

 

Синтаксис

object.AddNodeFromTree [sNameNode, [bAddChildAlso]]

Синтаксис свойства AddNodeFromTree состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.
sNameNode обязательное строковое выражение, определяющее код добавляемого узла.
 bAddChildAlso обязательное логическое выражение, определяющее признак добавления дочерных узлов sNameNode. По умолчанию принимает значение True, при котором вместе  с добавляемым узлом, добавляаются и его дочерные узли, если они существуют, в противном случае  добавляется только узел.

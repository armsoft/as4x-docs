Свойство isSelected

См. также  Пример  Применяется к

Возвращает признак того, выбран ли узел из динамического дерева.

Свойство для чтения.

 

Синтаксис

object.isSelected

Синтаксис свойства isSelected состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа динамическое дерево.

 

Тип данных

Логический

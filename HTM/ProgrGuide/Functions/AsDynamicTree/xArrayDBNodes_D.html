Свойство xArrayDBNodes

См. также  Пример  Применяется к

Возвращает ссылку типа XArrayDB на массив узлов динамического дерева.

 

Синтаксис

object.xArrayDBNodes [=sValue] 

Синтаксис свойства xArrayDBNodes состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
sValue строковое выражение, определяющее массив узлов динамического дерева типа XArrayDB.

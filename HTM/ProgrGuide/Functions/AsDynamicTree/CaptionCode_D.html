Свойство CaptionCode

См. также  Пример  Применяется к

Возвращает или устанавливает заголовок кода динамического дерева.

Свойство для чтения и записи.

 

Синтаксис

object.CaptionCode [=sValue] 

Синтаксис свойства CaptionCode состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
sValue строковое выражение, определяющее заголовок кода динамического дерева

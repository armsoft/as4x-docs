# Code հատկություն

[См. также](../AsDynamicTree.md) Օրինակ [Применяется к](../AsDynamicTree.md)

Վերադարձնում կամ նշանակում է դինամիկ ծառի կոդը։

## Շարահյուսություն

``` vb
object.Code [=sValue] 
```


Բաղադրիչներն են՝

    
| Պարամետր | Նկարագրություն |
|--|--|
| object | Դինամիկ ծառի հղում։ |
| sValue | նոր արժեք։ |

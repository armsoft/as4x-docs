Свойство MultiSelectMode

См. также  Пример  Применяется к

Возвращает или устанавливает признак множественного выбора из списка динамического дерева.

Свойство для чтения и записи.

 

Синтаксис

object.MultiSelectMode = [bMultiSelectValue]

Синтаксис свойства MultiSelectMode состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа динамическое дерево.
bMultiSelectValue логическое выражение, определяющее признак того, может ли выборка динамического дерева содержать несколько строк. Если задано значение False, то из динамического дерева может быть выбрана только одна строка. А если свойство установлено True, то можно также отмечать несколько строк.

 

Тип данных

Boolean

Примечание

Свойство MultiSelectMode актуальна, если значение свойства FullTree равноTrue, в противном случае MultiSelectMode не будет работать.

Свойство Left

См. также  Пример  Применяется к

Возвращает или устанавливает удаленность левого края динамического дерева от левого края окна приложения.

Свойство для чтения и записи.

 

Синтаксис

object.Left = [nLeftValue]

Синтаксис свойства Left состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа динамическое дерево.
nLeftValue численное выражение типа Single, определяющее расстояние между левым краем динамического дерева и левым краем окна приложения.

 

Тип данных

Single

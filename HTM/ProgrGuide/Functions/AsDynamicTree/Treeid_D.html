Свойство TreeId

См. также  Пример  Применяется к

Устонавливает значение идентификатора динамического дерева.

Свойство только для чтения.

 

Синтаксис

object.TreeId[sTreeId]

Синтаксис свойства TreeId состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
sTreeId строковое выражение, определяющее значение идентификатора динамического дерева

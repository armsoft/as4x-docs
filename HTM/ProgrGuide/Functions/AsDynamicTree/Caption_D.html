Свойство Caption

См. также  Пример  Применяется к

Возвращает или устанавливает значение заголовка динамического дерева.

Свойство для чтения и записи.

 

Синтаксис

object.Caption [=sValue] 

Синтаксис свойства Caption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
sValue строковое выражение, определяющее заголовок динамического дерева

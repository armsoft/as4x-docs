Свойство FullTree

См. также  Пример  Применяется к

Возвращает разрешимость выбора узлов любого уровня из динамического дерева.

Свойство для чтения.

 

Синтаксис

object.FullTree [=bValue] 

Синтаксис свойства FullTree состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа динамическое дерево.
bValue необязательное логическое выражение, определяющее разрешимость выбора узлов любого уровня из динамического дерева.

 

Тип данных

Логический

Объект DataSourceIndicator

Класс DataSourceIndicator предназначен для индикации процессинга источника данных из скрипта.

Свойства Описание
Properties

Возвращает ссылку на объект типа Dictionary. Данное свойство применяется для хранения некоторых значений для последующей их обработки.

 

Методы Описание
Init Инициализирует скриптовую процедуру из заданного модуля.
Run Выполняет скриптовую процедуру.

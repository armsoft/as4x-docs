Объект иерархический вспомогательный список

Создание экземпляра объекта типа иерархический вспомогательный список производится через объявление переменной:
    Dim VarName as New AsTreeModalBrowser
Иерархический вспомогательный список всегда основан на дереве.

Свойства Описание
Com

Возвращает комментарий выбранного элемента иерархического вспомогательного списка.
DocType Возвращает тип документа выбранного элемента иерархического вспомогательного списка.
ECom Возвращает комментарий на иностранном языке выбранного элемента иерархического вспомогательного списка.
Height Возвращает или устанавливает высоту окна иерархического вспомогательного списка.
ISN Возвращает внутрисистемный уникальный идентификатор документа выбранного элемента иерархического вспомогательного списка.
isSelected Возвращает признак того, выбрана ли строка из иерархического вспомогательного списка.
Leaf Возвращает признак того, является ли лепестком в дереве выбранный элемент иерархического вспомогательного списка.
Left Возвращает или устанавливает удаленность левого края иерархического вспомогательнoго списка от левого края окна  приложения.
Parent Возвращает код непосредственного родителя выбранного элемента иерархического вспомогательного списка.
Path Возвращает путь элемента в дереве, начиная с корня дерева, выбранного элемента иерархического вспомогательного списка.
RootNode Возвращает или устанавливает корневой элемент объекта типа иерархический вспомогательный список. Данное свойство служит для показа поддерьевьев.
StartUpAtCentre Возвращает или устанавливает признак показа иерархического вспомогательного списка в центре экрана.
Top Возвращает или устанавливает удаленность верхнего края иерархического вспомогательного списка от верхнего края окна приложения.
Value Возвращает ключ в дереве выбранного элемента иерархического вспомогательного списка.
Width Возвращает или устанавливает ширину окна иерархического вспомогательного списка.

 

Методы Описание
Show Активизирует, выводя на экран, иерархический вспомогательный список.

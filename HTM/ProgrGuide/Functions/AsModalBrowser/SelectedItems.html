Свойство SelectedItems

См. также  Пример  Применяется к

Возвращает коллекцию отобранных строк произвольного вспомогательного списка.

Свойство для чтения.

 

Синтаксис

object.SelectedItems

Синтаксис свойства SelectedItems состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа произвольный вспомогательный список.

 

Тип данных

Коллекция строк списка

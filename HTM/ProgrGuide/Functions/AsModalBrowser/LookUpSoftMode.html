Свойство LookUpSoftMode

См. также  Пример  Применяется к

Возвращает или устанавливает тип поиска значения для установки начальной позиции курсора при активизации произвольного вспомогательного списка.

Свойство для чтения и записи.

 

Синтаксис

object.LookUpSoftMode = [bLookUpSoftMode]

Синтаксис свойства LookUpSoftMode состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа произвольный вспомогательный список.
bLookUpSoftMode логическое выражение, определяющее признак префиксного (значение свойства True) или жесткого (значение свойства False) поиска значений. Префиксный поиск означает поиск значения в искомой колонке по префиксным начальным значениям. Жесткий поиск означает необходимость полного совпадения значения в искомой колонке. В результате нахождения значения в колонке курсор устанавливается на найденную строку.

 

Тип данных

Логический

Свойство Value

См. также  Пример  Применяется к

Возвращает значение указанной колонки из выбранной строки, в произвольном вспомогательном списке. Если, конечно, осуществлен выбор и значение свойства isSelected=True.

Свойство для чтения.

 

Синтаксис

object.Value (sColName)

Синтаксис свойства Value состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта произвольного вспомогательного списка.
sColName строковое выражение, определяющее идентификатор колонки из вспомогательного списка.

 

Тип данных

Variant

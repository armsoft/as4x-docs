Свойство Width

См. также  Пример  Применяется к

Возвращает или устанавливает ширину окна произвольного вспомогательного списка.

Свойство для чтения и записи.

 

Синтаксис

object.Width = [nWidthValue]

Синтаксис свойства Width состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа произвольный вспомогательный список.
nWidthValue выражение типа Single, определяющее ширину окна произвольного вспомогательного списка.

 

Тип данных

Single

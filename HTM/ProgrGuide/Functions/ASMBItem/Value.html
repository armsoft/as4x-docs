Свойство Value

См. также  Пример  Применяется к

Возвращает значение данной колонки. 

Свойство только для чтения.

 

Синтаксис

object.Value(sValue)

Синтаксис свойства Value состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта строка списка.
sValue выражение типа Variant, определяющее наименование или номер колонки.

 

Тип данных

Variant

Свойство ExistsControl

См. также  Пример  Применяется к

Определяется существование в диалоге элемента управления с указанным идентификатором.

Свойство для чтения.

 

Синтаксис

object.ExistControl(sControlName)

Синтаксис свойства ExistControl состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
sControlName строковое выражение, определяющее имя искомого элемента управления.

 

Тип данных

Логический

Свойство Value

См. также  Пример  Применяется к

Возвращает или устанавливает значение реквизита пользовательского диалога.

Свойство для чтения и записи.

 

Синтаксис

object.Value(vRekvName) = [vValue]

Синтаксис свойства Value состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
vRekvName выражение типа Variant, определяющее идентификатор реквизита диалога
vValue выражение типа Variant, определяющее значение реквизита в пользовательском диалоге

 

Примечание

Данное свойство при использовании можно опустить, например следующие примеры совершенно идентичны: xDialog.Value("BegDate") и xDialog("BegDate").

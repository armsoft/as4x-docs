Возможные значения атрибутов реквизита в обьекте диалог
 

Значение Описание
"B" отсутствует кнопка выбора значения реквизита (типа Folder, Tree, CH) из списка  
"C" после выбора значения реквизита (типа Folder или Tree) его наименование не приводится рядом с выбранным значением кода.
"D" данный реквизит доступен только для чтения
"F" признак использования альтернативного(русского) шрифта для колонки
"H" скрытый невидимый реквизит, при его наличии остальные атрибуты реквизита игнорируются.
"I" данный реквизит редактировать в режиме Overwrite вместо Insert (только для типа C(x))
"M" позволяет установить маску (шаблон ввода) для реквизитов длиной больше 12 (только для типа C(x)).
"R" данный реквизит обязателен для заполнения.
"U" преобразует латинские строчные буквы на прописные.
"S" текущее значение реквизита не запоминается в реестре

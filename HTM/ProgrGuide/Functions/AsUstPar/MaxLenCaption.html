Свойство MaxLenCaption

См. также  Пример  Применяется к

Возвращает максимальную длину заголовков реквизитов в пользовательском диалоге для данной страницы.

Свойство для чтения. Возвращаемое значение числовое.

 

Синтаксис

object.MaxLenCaption(iPageNum)

Синтаксис свойства MaxLenCaption состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
iPageNum численное выражение, определяющее номер страницы в пользовательском диалоге.

 

Тип данных

Целое число

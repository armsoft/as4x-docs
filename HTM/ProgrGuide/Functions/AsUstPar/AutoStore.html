Свойство AutoStore

См. также  Пример  Применяется к

Возвращает признак запоминания значений реквизитов диалога.

 

Синтаксис

call object.AutoStore( ) [=bValue]  

Синтаксис метода AutoStore состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.

 

Установки

Установки для bValue следующие:

Значение Описание
True Все реквизити диалога запоминаются в реестре
False Все реквизити диалога не запоминаются в реестре

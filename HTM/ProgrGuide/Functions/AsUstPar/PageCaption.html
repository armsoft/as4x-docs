Свойство PageCaption

См. также  Пример  Применяется к

Возвращает заголовок данной страницы в пользовательском диалоге.

Свойство для чтения.

 

Синтаксис

object.PageCaption(iPageNum).

Синтаксис свойства PageCaption состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
iPageNum численное выражение, определяющее номер страницы в пользовательском диалоге.

 

Тип данных

Строка

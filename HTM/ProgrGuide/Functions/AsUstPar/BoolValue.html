Свойство BoolValue

См. также  Пример  Применяется к

Возвращает признак изменения значения реквизита пользовательского диалога.

Свойство для чтения.

 

Синтаксис

object.BoolValue(vRekvName)

Синтаксис свойства BoolValue состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
vRekvName выражение типа Variant, определяющее идентификатор реквизита диалога

 

Тип данных

Логический

Свойство Cancel

См. также  Пример  Применяется к

Возвращает или устанавливает тип выхода из пользовательского диалога.

Свойство для чтения и записи.

 

Синтаксис

object.Cancel = [vValue]

Синтаксис свойства Cancel состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
vValue выражение логического типа, определяющее тип выход из диалога, с событием Cancel, т.е. кнопка OK не нажата.

 

Тип данных

Логический

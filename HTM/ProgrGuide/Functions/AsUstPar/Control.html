Свойство Control

См. также  Пример  Применяется к

Возвращает ссылку на реквизит пользовательского диалога.

Свойство для чтения.

 

Синтаксис

object.Control(RekvName).PropName

Синтаксис свойства Control состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
RekvName выражение типа Variant, определяющее идентификатор реквизита пользовательского диалога.
PropName строковое выражение, определяющее свойство реквизита. В зависимости от внутреннего типа реквизита становятся доступными различные наборы свойств.

 

Тип данных

Объект

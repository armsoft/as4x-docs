Свойство Top

См. также  Пример  Применяется к

Возвращает или устанавливает удаленность верхнего края диалога от верхнего края окна  приложения.

Свойство для чтения и записи.

 

Синтаксис

object.Top = [nTopValue]

Синтаксис свойства Top состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа диалог.
nTopValue выражение типа Single, определяющее расстояние между верхним краем диалога и верхним краем окна приложения.

 

Тип данных

Single

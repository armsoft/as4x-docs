Свойство PageCount

См. также  Пример  Применяется к

Возвращает количество страниц в пользовательском диалоге.

Свойство для чтения. Возвращаемое значение числовое.

 

Синтаксис

object.PageCount

Синтаксис свойства PageCount состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.

 

Тип данных

Целое число

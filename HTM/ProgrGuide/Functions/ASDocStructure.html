Объект структура документа

Данный объект делает доступной структуру документа из сценариев. Используя его можно определить свойства структурных элементов документа (реквизитов, грид-таблиц, колонок грид-таблиц).

Свойства Описание
Caption Возвращает или устанавливает новое значения заголовока документа.
EnglishCaption

Возвращает или устанавливает заголовок документа на English .
Element Возвращает элемент структуры документа  первого уровня.
ElementsCount Возвращает количество структурных элементов первого уровня.
NativeCaption

Возвращает или устанавливает заголовок документа на основном языке системы.
PageCaption

Меняет заголовок страницы документа перед его показом.

 

Методы Описание
Redraw Перерисовывает заголовока структурного элемента документа.

# Հաշվառում օբյեկտ

Հաշվառումը համակարում պետք է լինի նախօրոք նկարագրված և բեռնված։ [Նկարագրված հաշվառման](../Defs/Accounting.html) օբյեկտը ստացվում է [Accounting](Functions/SysDefManagment/Accounting.html) ֆունկցիայի միջոցով։ 

| Հատկություններ | Նկարագրություն |
|--|--|
| [AccFolder](AsAccounting/AccFolder.md) | Վերադարձնում է հաշիվների թղթապանակի ներքին անունը հաշվառումի նկարագրությունից։ |
| [Caption](AsAccounting/Caption.md) | Վերադարձնում է հաշվառման անվանումը։|
| [CountOperation](AsAccounting/CountOperation.md) | Վերադարձնում է նկարագրված գործողությունների քանակը հաշվառման մեջ։|
| [Cur](AsAccounting/Cur.md) | Վերադարձնում է հաշվառման բազմարժույթային լինելու հայտանիշը։|
| [Description](AsAccounting/Description.md) | Վերադարձնում է հաշվառման կրճատ բացատրական տեքստը նկարագրությունից։ |
| [FactHeader](AsAccounting/FactHeader.md) | Վերադարձնում կամ նշանակում է գործառնության նախնական ցուցադրման պատուհանի գլխագրի ձևանմուշը։ |
| [Limit](AsAccounting/Limit.md) | Վերադարձնում է ըստ հաշվառման օբյեկտների սահմանաչափի ավտոմատ ստուգուման հայտանիշը։ |
| [MainFolder](AsAccounting/MainFolder.md) | Վերադարձնում է հաշվառվող օբյեկների թղթապանակի ներքին անունը։ |
| [Name](AsAccounting/Name.md) | Վերադարձնում է հաշվառման ներքին անունը նկարագրությունից։ |
| [Remainder](AsAccounting/Remainder.md) | Վերադարձնում է հաշվառման մնացորդային լինելու հայտանիշը։ |
| [RemHeader](AsAccounting/RemHeader.md) | Վերադարձնում կամ նշանակում է գործառնության նախնական ցուցադրման պատուհանի մնացորդների ձևանմուշը։ |
| [RoundSumma](AsAccounting/RoundSumma.md) | Վերադարձնում է հաշվառման գումարի դրամային համարժեքի կլորացման աստիճանը։ |
| [RoundCurSumma](AsAccounting/RoundCurSumma.md) | Վերադարձնում է արժույթային գումարի կլորացման աստիճանը։ |


| Մեթոդներ | Նկարագրություն |
|--|--|
| [LastFactDate](AsAccounting/LastFactDate.md) | Վերադարձնում է վերջին գործառնության ամսաթիվը նշված հաշվառումից։ |

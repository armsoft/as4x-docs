Объект элемент структуры документа

Данный объект позволяет определить свойства элемента документа (реквизитов, грид-таблиц, колонок грид-таблиц).

Свойства Описание
Attributes Возвращает значение атрибута элемента документа.
Caption Возвращает или устанавливает новое значения заголовока-наименование элемента документа.
ElType Возвращает имя внутреннего типа данных элемента документа.
Kind Возвращает тип элемента структуры документа.
Name Возвращает идентификатор элемента структуры документа.
SubElement Возвращает элемент структуры документа второго уровня (колонки грид-таблицы).
SubElementsCount

Возвращает количество структурных элементов второго уровня (колонок грид-таблицы).

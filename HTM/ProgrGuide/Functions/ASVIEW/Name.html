Свойство Name

См. также  Пример  Применяется к

Возвращает идентификатор вида просмотра.

Свойство для чтения.

 

Синтаксис

object.Name

Синтаксис свойства Name состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра

 

Тип данных

Строка

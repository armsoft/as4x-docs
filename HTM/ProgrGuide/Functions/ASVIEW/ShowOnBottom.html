Свойство ShowOnBottom

См. также  Пример  Применяется к

Возвращает или устанавливает признак установки маркера на последней строчке при активизации вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.ShowOnBottom [=bValue] 

Синтаксис свойства ShowOnBottom состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
bValue логическое выражение, определяющее признак установки маркера на последней строчке вида просмотра.

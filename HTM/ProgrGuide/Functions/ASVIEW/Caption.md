# Caption հատկություն

[См. также](../Asview.md) [Օրինակ](../../Examples/E_AsView.html) [Применяется к](../Asview.md)

Վերադարձնում կամ նշանակում է դիտելու ձևի գլխագիրը։

## Շարահյուսություն

``` vb
object.Caption [=sValue] 
```
Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object | Դիտելու ձևի հղում։ |
| sValue | Նոր արժեք։ |

Свойство AllowVisa

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на визирование документа из вида просмотра. Чтобы значение True сработало необходимо наличие колонок с идентификатором fISN и fFOLDERID среди колонок источника данных.

Свойство для чтения и записи.

 

Синтаксис

object.AllowVisa [=bValue] 

Синтаксис свойства AllowVisa состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
bValue логическое выражение, разрешающее визирование документа

 

Установки

Установки для bValue следующие:

Значение Описание
False Не разрешать визирование документа
True Разрешать визирование документа

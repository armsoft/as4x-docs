Свойство UnMoveColumnCount

См. также  Пример  Применяется к

Возвращает или устанавливает количество неперемещаемых колонок в виде просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.UnMoveColumnCount [=nValue] 

Синтаксис свойства UnMoveColumnCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
nValue численное выражение целого типа, определяющее количество неперемещаемых колонок вида просмотра.

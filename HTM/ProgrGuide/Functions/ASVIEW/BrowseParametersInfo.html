Свойство BrowseParametersInfo

 См. также  Пример  Применяется к

Возвращает или устанавливает объект типа AsBrowserParametersInfo, в котором собержатся параметры диалога-фильтра вида просмора.

Свойство для чтения и записи.

 

Синтаксис

Set object.BrowseParametersInfo [=xValue] 

Синтаксис свойства BrowseParametersInfo состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
xValue объект типа AsBrowserParametersInfo.

Метод AppendToCondition

См. также  Пример  Применяется к

Добавляет условие к существующему условию фильтрации (Condition) вида просмотра.

 

Синтаксис

object.AppendToCondition sCondition, bAppendWithOr

Синтаксис метода AppendToCondition состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра.
sCondition строковое выражение, определяющее условие добавляемое к существующему.
bAppendWithOr необязательное логическое выражение, определяющее какой логической операцией объединять условия. По умолчанию принимает значение False, которое соответствует операции AND.

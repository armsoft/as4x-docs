Свойство AltFont

См. также  Пример  Применяется к

Возвращает или устанавливает признак использования альтернативного(русского) шрифта для колонки.

Свойство для чтения и записи.

 

Синтаксис

object.AltFont(sColName) [=bValue] 

Синтаксис свойства AltFont состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
sColName строковое выражение, определяющее наименование колонки.
bValue логическое выражение, принимающее значения 1 или 0, определяющее фонт колонки. При 1-е и значении параметра USEALTFONT=True для колонки используется альтернативный(русский) шрифт.

Свойство AllowHistory

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на просмотр журнала Log документа из вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.AllowHistory[ = bValue ]

Синтаксис свойства AllowHistory состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
bValue логическое выражение, разрешающее просмотр журнала Log документа.

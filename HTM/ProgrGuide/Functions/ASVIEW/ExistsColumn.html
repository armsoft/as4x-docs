Свойство ExistsColumn

См. также  Пример  Применяется к

Возвращает признак существования колонки в виде просмотре.

Свойство для чтения.

 

Синтаксис

object.ExistsColumn (sColName)

Синтаксис свойства ExistsColumn состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
sColName строковое выражение, являющееся идентификатором искомой колонки.

Свойство Seconds

 См. также  Пример  Применяется к

Возвращает или устанавливает промежуток времени по истеении которого агент вызывает процедуру.

Свойство для чтения и записи.

 

Синтаксис

object.Seconds = [sValue]

Синтаксис свойства Seconds состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа агент циклического вызова процедур.
sValue численное выражение, определяющее промежуток времени.

 

Тип данных

Длинное целое

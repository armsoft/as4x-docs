Свойство Text

 См. также  Пример  Применяется к

Возвращает или устанавливает текст полученной во время исполнения операции агентом циклического вызова процедур, и описывающее ход действий.

Свойство для чтения и записи.

 

Синтаксис

object.Text = [sValue]

Синтаксис свойства Text состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа агент циклического вызова процедур.
sValue строковое выражение, определяющее текст полученной во время исполнения операции агентом циклического вызова процедур.

 

Тип данных

Строка

Свойство Height

 См. также  Пример  Применяется к

Возвращает или устанавливает высоту модального окна для агента циклического вызова процедур.

Свойство для чтения и записи.

 

Синтаксис

object.Height = [nHeightValue]

Синтаксис свойства Height состоит из следующих астей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа агент циклического вызова процедур.
nHeightValue исленное выражение типа Single, определяющее высоту появляющегося окна.

 

Тип данных

Single

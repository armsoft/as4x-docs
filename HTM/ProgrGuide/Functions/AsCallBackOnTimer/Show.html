Метод Show

 См. также  Пример  Применяется к

Активизирует, выводя на экран, модальное окно для агента циклического вызова процедур.

 

Синтаксис

object.Show

Синтаксис свойства Show состоит из следующих астей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа агент циклического вызова процедур.

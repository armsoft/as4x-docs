Свойство Width

 См. также  Пример  Применяется к

Возвращает или устанавливает ширину модального окна для агента циклического вызова процедур.

Свойство для чтения и записи.

 

Синтаксис

object.Width = [nWidthValue]

Синтаксис свойства Width состоит из следующих астей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа агент циклического вызова процедур.
nWidthValue выражение типа Single, определяющее ширину окна появляющегося окна.

 

Тип данных

Single

Объект AsDataDesc (Динамический источник данных)

Создание объекта AsDataDesc обеспечивается через функцию CreateDataDesc.

Свойства Описание
ArrayDB Возвращает или устанавливает ссылку на массив значений колонок динамического источника данных.
Caption Возвращает или устанавливает заголовок динамического источника данных.
Columns Возвращает ссылку на массив, содержащий в себе информацию о добавляемых колонках.
DataIndicate Возвращает или устанавливает внешний вид индикации открытия динамического источника данных.
DataMode Возвращает или устанавливает источник значений добавляемых колонок.
DefaultFunction Устанавливает имя контекстно-вызываемой процедуры при двойном щелчке мыши либо при нажатии клавиши Enter.
ECaption Возвращает или устанавливает заголовок динамического источника данных на иностранном языке.
HeadLinesCount Возвращает или устанавливает ширину заголовков колонок.
IndexOfFolderId Возвращает или устанавливает номер колонки с полем fFOLDERID.
IndexOfIsn Возвращает или устанавливает номер колонки с полем fISN.
Name Возвращает или устанавливает имя динамического источника данных.
Param Устанавливает значение параметра по индексу или коду.
Parameters Возвращает или устанавливает ссылку на массив, содержащий в себе информацию о добавляемом параметре.
RegistrFunctions Возвращает или устанавливает ссылку на массив, содержащий в себе информацию о добавляемых функциях.
SQL Возвращает или устанавливает запрос выборки определенных строк для динамического источника данных.
UpdateSQL Возвращает или устанавливает обновлние SQL запроса для выборки нужных строк.

 

Методы Описание
AddColumn Добавляет колонку для динамического источника данных.
AddParam Добавляет параметр для динамического источника данных.
GetParINdex Возвращает индекс параметра по заданному имени.
RegistrFunction Добавляет в контекстное меню вызов пользовательской функции.
RegistrNode Добавляет узел в дерево вызовов пользовательских функций.
RemoveColumn Удаляет колонку из данного вида динамического источника данных.
RemoveParam Удаляет параметр из динамического источника данных.

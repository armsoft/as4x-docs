Объект AsBrowserParametersInfo (параметры диалога-фильтра вида просмора).

Объект AsBrowserParametersInfo содержит в себе параметры диалога-фильтра вида просмора. Создание данного объекта обеспечивается через метод CreateBrowseParametersInfo  объекта пользовательского диалога.

 

Свойства Описание
DialogName

Возвращает или устанавливает имя диалога-филтра вида просмора.
DialogValue

Возвращает значение параметра из диалога.
DialogValueItem

Возвращает признак наличия  параметра диалога в списке выбранных значений.
GetDialogParCount

Возвращает количество параметров диалога-филтра вида просмора.
HasParameter

Возвращает признак существования заданного параметра в диалоге.
Name

Возвращает или устанавливает имя AsBrowserParametersInfo.


 

Методы Описание
AddDialogParameter

Создает параметр диалога-фильтра вида просмора.
AddDSParameter

Создает параметр для источника данных вида просмора.
GetBrowseParametersReportt

Создает объект типа отчет, в котором содержится информация о параметрах  диалога-фильтра вида просмора.

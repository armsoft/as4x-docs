Свойство UseStyles

См. также  Пример  Применяется к

Возвращает признак  использования описанных форматов для ячеек таблицы шаблона печати типа .xlsx.

Синтаксис

object.UseStyles[= value]

Синтаксис свойства UseStyles состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа TemplateSubstitutionGrid.
value логическое выражение, oпределяющее признак использования описанных форматов ячеек таблицы шаблона печати. Принимает значения True или False. При значении True разрешается использование описанных форматов ячеек, при значении False - не разрешается.

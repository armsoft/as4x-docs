Метод Image

См. также  Пример  Применяется к

Возвращает подстановку изображения по имени.

 

Синтаксис

object.Image(Name)

Синтаксис метода Image состоит из следующих частей:

Параметр Описание
Name строковое выражение, определяющее имя подстановки изображения

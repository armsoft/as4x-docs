Свойство Value

См. также  Пример  Применяется к

Возвращает или устанавливает значение элемента табличной подстановки.

Свойство для чтения и записи.

 

Синтаксис

object.Value(Row, Column)

Параметр Описание
Row численное выражение целого типа, определяющее строку в табличной подстановке.
Column строковое выражение, определяющее колонку в  зтабличной подстановке.


Тип данных

Строка

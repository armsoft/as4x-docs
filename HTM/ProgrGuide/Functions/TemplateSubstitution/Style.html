Свойство Style

См. также  Пример  Применяется к

Возвращает или устанавливает формат конкретной ячейки таблицы шаблона печати типа .xlsx по заданной строке и колонке.

Синтаксис

object.Style(row, column)[=value]

Синтаксис свойства Style состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа TemplateSubstitutionGrid.
row численное выражение типа Long, oпределяющее строку ячейки.
column строковое выражение, oпределяющее имя колонки шаблона печати.
value строковое выражение, oпределяющее ссылку на экземпляр объекта типа TemplateSubstCellStyle.

# RowCount հատկություն

Տես նաև Օրինակ [Կիրառվում է](../TemplateSubstitutionGrid.md)


Վերադարձնում է աղյուսակային տեղադրման արժեքի տողերի քանակը։
Возвращает количество сторок в табличной подстановке.

## Շարահյուսություն

```as4x
object.RowCount
```

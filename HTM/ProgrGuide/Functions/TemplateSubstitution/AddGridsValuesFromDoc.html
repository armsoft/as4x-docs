Метод AddGridsValuesFromDoc

См. также  Пример  Применяется к

Для всех грид-таблиц документа создаются табличные подстановки. Созданные табличные подстановки имеют те же имена и содержат те же данные.

 

Синтаксис

object.AddGridsValuesFromDoc(Doc, [Prefix], [Postfix])

Синтаксис метода AddGridsValuesFromDoc состоит из следующих частей:

Параметр Описание
Doc объект типа документ
Prefix необязательное строковое выражение, определяющее префиксное значение, добавляемое спереди имен табличных подстановок
Postfix необязательное строковое выражение, определяющее постфиксное значение, добавляемое сзади имен табличных подстановок

Метод Grid

См. также  Пример  Применяется к

Возвращает табличную подстановку по имени.

Синтаксис

object.Grid(Name)

Синтаксис метода Grid состоит из следующих частей:

Параметр Описание
Name строковое выражение, определяющее имя табличной подстановки.

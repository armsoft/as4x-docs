Свойство FormatedValue

См. также  Пример  Применяется к

Возвращает символьное значение элемента табличной подстановки.

Свойство только для чтения.

 

Синтаксис

object.FormatedValue(Row, Column)

Синтаксис свойства FormatedValue состоит из следующих частей:

Параметр Описание
Row численное выражение целого типа, определяющее строку в табличной подстановке.
Column строковое выражение, определяющее колонку в  зтабличной подстановке.


Тип данных

Строка

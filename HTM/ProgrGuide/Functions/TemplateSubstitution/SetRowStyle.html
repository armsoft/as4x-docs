Метод SetRowStyle

См. также  Пример  Применяется к

Задает форматы ячеек строки таблицы шаблона печати типа .xlsx..

Синтаксис

object.SetRowStyle[row, style]

Синтаксис метода SetRowStyle состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа TemplateSubstitutionGrid.
row численное выражение типа Long, oпределяющее строку ячейки таблицы шаблона печати.
style строковое выражение, oпределяющее ссылку на экземпляр объекта типа TemplateSubstCellStyle.

Метод AddRekvValuesFromDoc

См. также  Пример  Применяется к

Добавляет значения всех реквизитов документа в качестве атомарных подстановок.

 

Синтаксис

object.AddRekvValuesFromDoc(Doc, [Prefix], [Postfix])

Синтаксис метода AddRekvValuesFromDoc состоит из следующих частей:

Параметр Описание
Doc объект типа документ
Prefix необязательное строковое выражение, определяющее префиксное значение, добавляемое спереди имен реквизитов документа
Postfix необязательное строковое выражение, определяющее постфиксное значение, добавляемое сзади имен реквизитов документа

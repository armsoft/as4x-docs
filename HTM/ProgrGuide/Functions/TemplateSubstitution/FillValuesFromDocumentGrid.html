Метод FillValuesFromDocumentGrid

См. также  Пример  Применяется к

Заполняет табличную подстановку из грид-таблицы документа.

 

Синтаксис

object.FillValuesFromDocumentGrid(Grid)

Синтаксис метода FillValuesFromDocumentGrid состоит из следующих частей:

Параметр Описание
Grid объект типа AsGrid, явлется источником данных для табличной подстаноки.

Метод Add

См. также  Пример  Применяется к

Добавляет атомарную подстановку.

 

Синтаксис

object.Add(Code,Value,[Length])

Синтаксис метода Add состоит из следующих частей:

Параметр Описание
Code строковое выражение, определяющее имя атомарной подстановки.
Value строковое выражение, определяющее значение атомарной подстановки.
Length необязательное численное выражение целого типа, определяющее уровень длину подстановки.

Свойство Param

 См. также  Пример  Применяется к

Устанавливает значение по умолчанию параметра по заданному индексу или коду.

Свойство для чтения и записи.

 

Синтаксис

object.Param [CodeOrIndex] [=vValue]  

Синтаксис свойства Param состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
CodeOrIndex выражение типа вариант, определяющее код или индекс параметра.
vValue выражение типа вариант, определяющее значение по умолчанию параметра.

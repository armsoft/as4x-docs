Свойство DataIndicate

См. также  Пример  Применяется к

Возвращает или устанавливает внешний вид индикации открытия динамического источника данных.

Свойство для чтения.

 

Синтаксис

object.DataIndicate [=iValue] 

Синтаксис свойства DataIndicate состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа динамическое дерево.
iValue необязательное численное выражение, принимаюшее значения 0 (появляются только песочные часы), 1 (появляется индикатор продвижения выполнения запроса) или 2 (отсутствуют песочные часы и индикатор).

Свойство Columns

См. также  Пример  Применяется к

Возвращает ссылку на массив, содержащий в себе информацию о добавляемых колонках.

Свойство для чтения.

 

Синтаксис

object.Columns

Синтаксис свойства Columns состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.

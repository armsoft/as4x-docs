Свойство Parameters

 См. также   Пример  Применяется к

Возвращает ссылку на массив, содержащий в себе информацию о добавляемом параметре.

Свойство для чтения.

 

Синтаксис

object.Parameters

Синтаксис свойства Parameters состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.

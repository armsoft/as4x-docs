Свойство ECaption

См. также  Пример  Применяется к

Возвращает или устанавливает заголовок динамического источника данных на иностранном языке.

Свойство для чтения и записи.

 

Синтаксис

object.ECaption [=sValue] 

Синтаксис свойства ECaption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
sValue строковое выражение, определяющее заголовок динамического источника данных на иностранном языке

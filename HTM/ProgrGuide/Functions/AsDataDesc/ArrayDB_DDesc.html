Свойство ArrayDB

См. также  Пример  Применяется к

Возвращает или устанавливает ссылку на массив значений колонок динамического источника данных.

Свойство для чтения и записи.

 

Синтаксис

object.ArrayDB [=sValue] 

Синтаксис свойства ArrayDB состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
sValue строковое выражение, определяющее массив значений колонок динамического источника данных, ссылку на объект которого нужно возвращать

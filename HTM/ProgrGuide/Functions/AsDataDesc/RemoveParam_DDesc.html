Метод RemoveParam

См. также  Пример  Применяется к

Удаляет параметр из динамического источника данных.

Свойство для чтения.

 

Синтаксис

object.RemoveParam [CodeOrIndex]

Синтаксис метода RemoveParam состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.
CodeOrIndex обязательное выражение типа вариант, определяющее код или индекс удаляемого параметра.

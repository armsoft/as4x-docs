Свойство IndexOfFolderId

 См. также   Пример  Применяется к

Возвращает или устанавливает номер колонки с полем fFOLDERID.

Свойство для чтения.

 

Синтаксис

object.IndexOfFolderId[=iValue] 

Синтаксис свойства IndexOfFolderId состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
iValue численное выражение, определяющее номер колонки с полем fFOLDERID. По умолчанию установлено значение -1.

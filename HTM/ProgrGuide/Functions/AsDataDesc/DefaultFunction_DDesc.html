Свойство DefaultFunction

См. также  Пример  Применяется к

Устанавливает имя контекстно-вызываемой процедуры при двойном щелчке мыши либо при нажатии клавиши Enter.

Свойство для чтения и записи.

 

Синтаксис

object.DefaultFunction [=sValue] 

Синтаксис свойства DefaultFunction состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.
sValue строковое выражение, определяющее имя контекстно-вызываемой процедуры.

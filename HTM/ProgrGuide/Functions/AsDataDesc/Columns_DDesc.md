# Columns հատկություն

[См. также](../AsDataDesc.md) Օրինակ [Применяется к](../AsDataDesc.md) 

Վերադարձնում է ավելացվող սյունակների մասիվը։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.Columns
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
|  object  | Դինամիկ տվյալների աղբյուրի հղում։ |

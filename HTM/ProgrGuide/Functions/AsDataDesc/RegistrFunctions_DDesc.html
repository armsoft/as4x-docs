Свойство RegistrFunctions

 См. также   Пример  Применяется к

Возвращает ссылку на массив, содержащий в себе информацию о добавляемых функциях.

Свойство для чтения.

 

Синтаксис

object.RegistrFunctions

Синтаксис свойства RegistrFunctions состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.

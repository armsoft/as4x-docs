Свойство IndexOfIsn

 См. также   Пример  Применяется к

Возвращает или устанавливает номер колонки с полем fISN.

Свойство для чтения.

 

Синтаксис

Синтаксис свойства IndexOfIsn [=iValue] 

состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
iValue численное выражение, определяющее номер колонки с полем fISN. По умолчания установлено значение -1.

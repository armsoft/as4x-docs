Свойство LookUpColumn

 См. также  Пример  Применяется к

Возвращает или устанавливает имя той колонки, по которой производится поиск значения при активизации произвольного вспомогательного списка.

Свойство для чтения и записи.

 

Синтаксис

object.LookUpColumn = [sLookUpColumnValue]

Синтаксис свойства LookUpColumn состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция строк списка.
sLookUpColumnValue строковое выражение, определяющее имя колонки по которой производится поиск значения для установки начальной позиции курсора при активизации произвольного вспомогательного списка.

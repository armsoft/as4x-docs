Свойство Item

 См. также  Пример  Применяется к

Возвращает элемент из коллекции строк списка по ключу.

Свойство только для чтения.

 

Синтаксис

object.Item (vNameMBItem)

Синтаксис свойства Item состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция строк списка.
vNameMBItem выражение типа Variant, определяющее ключ элемента в коллекции. Ключом элемента может быть или порядковый номер элемента в коллекции, или значение ключевой колонки для строки списка.

 

Тип данных

Строка списка

Объект TemplateSubstitution

Объект TemplateSubstitution предназначен для описания подстановок в шаблоны печати. В подстановки могут быть осуществлены в следущие типы шаблонов:
        0     -    Microsoft Word  (*.doc)
        1    -    Microsoft Excel (*.xls)
        2    -    MS Word Open XML (*.docx)
        3    -    MS Excel Open XML (*.xlsm)
        4    -    PDF (*.pdf)
        5    -    Open Document Text (*.odt)
        6    -    Open Document Spreadsheet (*.ods)

Методы Описание
Add Добавляет атомарную подстановку.
AddGrid

Добавляет табличную подстановку.
AddGridValuesFromDoc

Добавляет табличную подстановку и копирует в нее данные и грид-таблицы документа.
AddGridsValuesFromDoc Для всех грид-таблицы документа создаются табличные подстановки. Созданные табличные подстановки имеют те же именна и содержат то же данные.
AddImage Добаляет подстановку изображения.
AddRekvValuesFromDoc

Добавляет значения всех реквизитов документа в качестве атомарных подстановок.
ExistsGrid

Поверяет существование табличной подстановки.
ExistsImage Поверяет существование подстановки изображения.
Grid Возвращает табличную подстановку по имени.
Image Возвращает подстановку изображения по имени.
SubstituteToTemplate Осуществляет подстановку в шаблон печати.
SubstituteToTemplateAndShow Осуществляет подстановку в шаблон печати и показывает результат.

Свойство PivotRows

См. также  Пример  Применяется к

Возвращает список, состоящее из транспонированных строк.

Синтаксис

object.PivotRows

Синтаксис свойства PivotRows состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа транспонирование данных.

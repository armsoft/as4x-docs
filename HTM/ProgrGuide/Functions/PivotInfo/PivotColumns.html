Свойство PivotColumns

См. также  Пример  Применяется к

Возвращает список транспонированных колонок.

 

Синтаксис

object.PivotColumns

Синтаксис свойства PivotColumns состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа транспонирование данных.

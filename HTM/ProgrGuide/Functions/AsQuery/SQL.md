# SQL հատկություն

[Օրինակ](../../Examples/E_AsQuery.md) [Կիրառվում է](../AsQuery.md)

Վերադարձնում կամ նշանակում է SQL հարցման տեքստը։ 

## Շարահյուսություն

```vb
object.SQL [= sSQL]
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | AsQuery օբյեկտի հղում։ |
| sSQL | Հարցման տեքստ։ |

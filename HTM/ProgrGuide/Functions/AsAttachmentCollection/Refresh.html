Метод Refresh

 См. также  Пример  Применяется к

Обновляет коллекцию файловых дополнений, загружая их из сервера.

 

Синтаксис

object.Refresh

Синтаксис метода Refresh состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция файловых дополнений.

Метод Add

 См. также  Пример  Применяется к

Добавляет новый элемент в коллекцию файловых дополнений.

 

Синтаксис

object.Add (sPath, [sComment])

Синтаксис метода Add состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция файловых дополнений.
sPath строковое выражение, определяющее полный путь добавляемого файлового дополнения.
sComment строковое выражение, определяющее комментарий для добавляемого файла.

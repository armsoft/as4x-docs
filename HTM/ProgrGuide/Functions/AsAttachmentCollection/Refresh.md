# Refresh մեթոդ

[См. также](../AsAttachmentCollection.md) Օրինակ [Применяется к](../AsAttachmentCollection.md)

Թարմացնում է հավաքածուն բեռնելով տվյալների պահոցից։

## Շարահյուսություն 

``` vb
object.Refresh
```
Բաղադրիչներն են՝


| Պարամետր | Նկարագրոթյոն |
|--|--|
| object | Փաստաթղթի կցված ֆայլերի հավաքածուի հղում։ |

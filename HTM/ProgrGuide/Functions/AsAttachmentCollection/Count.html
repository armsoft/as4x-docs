Свойство Count

 См. также  Пример  Применяется к

Возвращает количество элементов в коллекции файловых дополнений.

Свойство только для чтения.

 

Синтаксис

object.Count

Синтаксис свойства Count состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция файловых дополнений.

 

Тип данных

Целое число

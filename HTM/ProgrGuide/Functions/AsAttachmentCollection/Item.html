Свойство Item

 См. также  Пример  Применяется к

По имени или номеру возвращает элемент из коллекции файловых дополнений.

Свойство только для чтения.

 

Синтаксис

object.Item (vNameAttachment)

Синтаксис свойства Item состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция файловых дополнений.
vNameAttachment выражение типа Variant, определяющее имя или номер в коллекции файлового дополнения.

 

Тип данных

Файловое дополнение

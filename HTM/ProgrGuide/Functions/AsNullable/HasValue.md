# HasValue հատկություն 

[Տես նաև](../AsNullable/Value.md) Օրինակ [Կիրառվում է](../AsNullable.md)

``` vb

```

| Պարամետր | Նկարագրություն |

| object | AsNullable օբյեկտի հղում։|

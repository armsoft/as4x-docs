# ExportToExcel մեթոդ


[См. также](../Asdoc.md) Օրինակ [Применяется к](../Asdoc.md)
  
Փաստաթղթի պարունակությունը արտահանում է Excel։

## Շարահյուսություն


``` vb
object.ExportToExcel
```
Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Փաստաթուղթ օբյեկտի հղում։|

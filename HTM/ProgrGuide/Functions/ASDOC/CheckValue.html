Свойство CheckValue

См. также  Пример  Применяется к

Возвращает или устанавливает режим проверки типа реквизита документа при присвоении значения. Удобно для присвоения значения реквизитам загруженного документа избегая проверок типов.

Свойство для чтения и записи.

 

Синтаксис

object.CheckValue(RekvName)[=bValue]

Синтаксис свойства CheckValue состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
RekvName строковое выражение, определяющее идентификатор реквизита документа
bValue логическое выражение, определяющее необходимость проверки значения реквизита

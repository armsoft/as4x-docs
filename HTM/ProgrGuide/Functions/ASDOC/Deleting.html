Свойство Deleting

См. также  Пример  Применяется к

Возвращает значение True если документ находится в процессе удаления, и False в противном случае.

Свойтво только для чтения

 

Синтаксис

object.Deleting

Синтаксис свойства Deleting состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Логический

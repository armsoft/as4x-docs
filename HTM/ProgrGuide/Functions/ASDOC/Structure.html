Свойство Structure

См. также  Пример  Применяется к

Возвращает ссылку на объект типа структура документа.  Свойство используется для определения свойств структурных элементов документа (реквизитов, грид-таблиц, колонок грид-таблиц).

Свойство для чтения.

 

Синтаксис

object.Structure

Синтаксис свойства Structure состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Структура документа.

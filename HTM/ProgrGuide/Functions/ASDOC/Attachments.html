Свойство Attachments

 См. также  Пример  Применяется к

Возвращает коллекцию файловых дополнений, прикрепленных к данному документу.

Свойство для чтения.

 

Синтаксис

object.Attachments

Синтаксис свойства Attachments состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Коллекция файловых дополнений

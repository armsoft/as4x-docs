Свойство Body

См. также Пример Применяется к

Возвращает тело документа DOCS таблицы без гридов в формате rekvizit1:rekvalue1 rekvizit2:rekvalue2 ...

Синтаксис

object.Body()

Синтаксис свойства Body состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

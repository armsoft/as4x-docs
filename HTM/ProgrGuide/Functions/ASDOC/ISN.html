Свойство ISN

См. также  Пример  Применяется к

Возвращает внутрисистемный уникальный идентификатор документа.

Свойство только для чтения.

 

Синтаксис

object.ISN

Синтаксис свойства ISN состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Длинное целое

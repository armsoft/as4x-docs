Свойство IsNew

См. также  Пример  Применяется к

Определяет является ли документ новым (не сохраненным в базе данных) или черновиком.

Свойство только для чтения.

 

Синтаксис

object.IsNew

Синтаксис свойства IsNew состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Логический

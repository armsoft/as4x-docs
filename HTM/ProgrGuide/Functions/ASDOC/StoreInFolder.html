Метод StoreInFolder

 См. также  Пример  Применяется к

Сохраняет элемент папки.

 

Синтаксис

object.StoreInFolder ( xFoldEl )

Синтаксис метода StoreInFolder состоит из следующих астей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
xFoldEl переменная, ссылающаяся на объект типа элемент папки. Он должен быть заранее создан и инициализирован, потом только сохранен.

# CreateDialog մեթոդ

[См. также](../Asdoc.md) Օրինակ [Применяется к](../Asdoc.md)

Ստեղծում է երկխոսության պատուհան, որի մեջ մտնում են փաստաթղթի  բոլոր տեսանելի դաշտերը։

## Շարահյուսություն

``` vb
object.CreateDialog
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object  | Փաստաթուղթ օբյեկտի հղում։ |

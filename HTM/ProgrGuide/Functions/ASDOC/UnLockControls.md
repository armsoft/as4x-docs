# UnLockControls մեթոդ

[См. также](../Asdoc.md) Օրինակ [Применяется к](../Asdoc.md)

Փաստաթղթի բոլոր դաշտերը դարձնում է  խմբագրելի։



## Շարահյուսություն

``` vb
object.UnLockControls()
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object | Փաստաթուղթ օբյեկտի հղում։ |

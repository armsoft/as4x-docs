Свойство Image

См. также  Пример  Применяется к

Возвращает или устанавливает значение реквизита-изображения документа.

Свойство для чтения и записи.

 

Синтаксис

object.Image(RekvName)[=Value]  

Синтаксис свойства Image состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
RekvName выражение типа Variant, определяющее идентификатор реквизита-изображения.
Value выражение типа Variant, определяющее подставляемое значение реквизита-изображения.

 

Примечание

При помощи данного свойства можно значение реквизита-изображения из одного документа скопировать в другой документ.

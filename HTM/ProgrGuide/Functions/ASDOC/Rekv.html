Свойство Rekv

См. также  Пример  Применяется к

Возвращает или устанавливает значение реквизита документа.

Свойство для чтения и записи. Свойство по умолчанию.

 

Синтаксис

object.Rekv(RekvName)[=sValue]  

Синтаксис свойства Rekv состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
RekvName выражение типа Variant, определяющее идентификатор реквизита документа
sValue выражение типа Variant, определяющее подставляемое значение реквизита

 

Примечание

Данное свойство является свойством по умолчанию, что означает возможность опускания ключевого слова Rekv при его использовании. Т.е. записи doc.Rekv("ACCDB") и doc("ACCDB") эквивалентны.

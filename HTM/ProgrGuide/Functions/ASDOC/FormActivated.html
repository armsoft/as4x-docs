Свойство FormActivated

См. также  Пример  Применяется к

Свойство FormActivated принимает значение True если активизирована UI документа. Свойство имеет смысл применять в подпрограммах которые могут быть вызваны как при визуальной , так и при фоновой обработке обработке документа, для разграничения этих случаев.

Синтаксис

object.FormActivated

Синтаксис метода FormActivated состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

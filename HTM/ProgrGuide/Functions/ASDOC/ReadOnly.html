Свойство ReadOnly

См. также  Пример  Применяется к

Возвращает или устанавливает статус режима редактирования реквизита документа.

Свойство для чтения и записи.

 

Синтаксис

object.ReadOnly(RekvName)[=bValue]  

Синтаксис свойства ReadOnly состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
RekvName строковое выражение, определяющее идентификатор реквизита документа
bValue логическое выражение, определяющее возможность редактирования указанного реквизита

 

Установки

Установки для bValue следующие:

Значение Описание
True Редактирование реквизита допустимо
False Редактирование реквизита недопустимо

Свойство ExistsSub

См. также  Пример  Применяется к

Возвращает признак наличии данной процедуры в описании документа.

Свойство для чтения.

 

Синтаксис

object.ExistsSub (sSubName)

Синтаксис свойства ExistsSub состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
sSubName строковое выражение, определяющее наименование процедуры

 

Тип данных

Логическое

Метод StoreFact

См. также  Пример  Применяется к

Сохраняет проводку. Данный метод нужно вызывать из обработика системного события Action.

 

Синтаксис

object.StoreFact ( xFact )

Синтаксис метода StoreFact состоит из следующих астей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
xFact переменная, ссылающаяся на объект типа проводка. Он должен быть заранее создан и инициализирован, потом только сохранен.

# FormActivated հատկություն

[См. также](../Asdoc.md) Օրինակ [Применяется к](../Asdoc.md)

Ցույց է տալիս փաստաթղթի պատուհանի ակտիվացված լինելը։ 

## Շարահյուսություն

``` vb
object.FormActivated
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Փաստաթուղթ օբյեկտի հղում։ |

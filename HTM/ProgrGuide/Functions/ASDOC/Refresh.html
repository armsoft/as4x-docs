Метод Refresh

См. также  Пример  Применяется к

Обновляет значение реквизита на форме документа.

 

Синтаксис

object.Refresh(RekvName)

Синтаксис метода Refresh состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
RekvName выражение типа Variant, определяющее идентификатор реквизита документа

 

Примечание

Метод Refresh часто используется для обновления значений одних реквизитов, в зависимости от введенных значений других.

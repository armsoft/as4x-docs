Свойство IsHiddenGrid

См. также Пример Применяется к

Возвращает признак невидимости грид-таблицы документа.


Синтаксис

object.IsHiddenGrid(sGridName)

Синтаксис свойства IsHiddenGrid состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
sGridName строковое выражение, определяющее наименование грид-таблицы документа


Тип данных

Логический

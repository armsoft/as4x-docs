Метод RefreshTS

См. также Пример Применяется к

Обновляет отметку последнего изменения документа.


Синтаксис

object.RefreshTS

Синтаксис свойства RefreshTS состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

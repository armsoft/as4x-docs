Свойство ReadOnlyIm

См. также  Пример  Применяется к

Возвращает или устанавливает статус режима редактирования реквизита-изображения документа.

Свойство для чтения и записи.

 

Синтаксис

object.ReadOnlyIm(RekvName)[=bValue]  

Синтаксис свойства ReadOnlyIm состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
RekvName строковое выражение, определяющее идентификатор реквизита-изображения документа
bValue логическое выражение, определяющее возможность редактирования указанного реквизита-изображения

 

Установки

Установки для bValue следующие:

Значение Описание
True Редактирование реквизита-изображения допустимо
False Редактирование реквизита-изображения недопустимо

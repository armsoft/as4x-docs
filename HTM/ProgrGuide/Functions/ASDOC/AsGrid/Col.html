Свойство Col

См. также  Пример  Применяется к

Возвращает номер текущей колонки в грид таблице документа.

Свойство для чтения.

 

Синтаксис

object.Col

Синтаксис свойства Col состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

 

Примечание

Индекс возвращаемого значения начинается с 0 до ColCount-1.

Тип данных

Целое число

Метод RemoveRow

См. также  Пример  Применяется к

Удаляет указанную строку из грид таблицы документа.

 

Синтаксис

object.RemoveRow (index)

Синтаксис метода RemoveRow состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе
index выражение типа Variant, определяющее номер удаляемой строчки. Нумерация строк начинается с 0.

Свойство AllowDelete

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на интерактивное удаление строчки из грид-таблицы.

Свойство для чтения и записи.

 

Синтаксис

object.AllowDelete [= bValue]

Синтаксис свойства AllowDelete состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе
bValue логическое выражение, определяющее разрешение на интерактивное удаление строки из грид-таблицы.

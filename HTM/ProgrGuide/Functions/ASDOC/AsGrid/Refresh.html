Метод Refresh

См. также  Пример  Применяется к

Обновляет содержание грид-таблицы на экране.

 

Синтаксис

object.Refresh

Синтаксис метода Refresh состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

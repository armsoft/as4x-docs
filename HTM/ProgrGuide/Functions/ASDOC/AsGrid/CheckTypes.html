Свойство CheckTypes

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на проверку внутренних типов данных колонок грид-таблицы.

Свойство для чтения и записи.

 

Синтаксис

object.CheckTypes= [value]

Синтаксис свойства  CheckTypes состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе
value логическое выражение, определяющее разрешение на прверку внутренных типов данных колонок грид-таблицы. Если установлено значение True, то проверяются типы всех колонок грид-таблицы, в противном случае не проверяются.

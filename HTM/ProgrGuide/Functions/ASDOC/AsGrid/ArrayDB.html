Свойство ArrayDB

См. также  Пример  Применяется к

Возвращает ccылку на массив значений грид таблицы. Для отображения измененных значений элементов массива на экране необходимо применить метод Refresh.

Свойство для чтения.

 

Синтаксис

set sArrDB = object.ArrayDB

Синтаксис свойства ArrayDB состоит из следующих частей:

Параметр Описание
sArrDB переменная, которая получает ссылку на массив значений грид-таблицы.
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

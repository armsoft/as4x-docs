Свойство NumFromName

См. также  Пример  Применяется к

Возвращает порядковый номер колонки грида по идентификатору колонки. Если в грид-таблице отсутствует колонка с заданным идентификатором или же номером колонки, то возвращается -1. Задание в качестве аргумента номера колонки, удобно для проверки существования колонки с таким номером.

Свойство для чтения.

 

Синтаксис

object.NumFromName(ColName)

Синтаксис свойства NumFromName состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе
ColName выражение типа Variant, определяющее номер или идентификатор колонки в грид-таблице. Причем первая колонка имеет индекс 0.

 

Тип данных

Целое число

Свойство Name

См. также  Пример  Применяется к

Возвращает наименование грид таблицы документа в верхнем регистре(UCASE).

Свойство для чтения.

 

Синтаксис

object.Name

Синтаксис свойства Name состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

 

Тип данных

Строка

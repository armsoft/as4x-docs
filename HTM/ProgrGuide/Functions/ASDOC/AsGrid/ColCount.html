Свойство ColCount

См. также  Пример  Применяется к

Возвращает количество колонок в грид-таблице документа.

Свойство для чтения.

 

Синтаксис

object.ColCount

Синтаксис свойства ColCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

 

Тип данных

Целое число

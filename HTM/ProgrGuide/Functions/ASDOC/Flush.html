Метод Flush

См. также  Пример  Применяется к

Oсуществляет сброс тела документа в БД , в результате чего обновляется запись данного документа только в таблице DOCS. Вызов метода возможен только из события Action.

 

Синтаксис

object.Flush

Синтаксис метода Flush состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

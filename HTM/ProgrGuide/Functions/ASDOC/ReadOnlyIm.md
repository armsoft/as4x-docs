# ReadOnlyIm հատկություն 

[См. также](../Asdoc.md) Օրինակ [Применяется к](../Asdoc.md)

Վերադարձնում կամ նշանակում է փաստաթղթի նկար-դաշտի  խմբագրելիության հատկանիշը։

## Շարահյուսություն

``` vb
object.ReadOnlyIm(RekvName)[=bValue]  
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object | Փաստաթուղթ օբյեկտի հղում։ |
| rekvName | Նկար-դաշտի ներքին անուն։|
| bValue | Նոր արժեք։ |


## Արժեքներ


| Արժեքներ | Նկարագրություն |
|--|--|
| True |Նկար-դաշտի խմբագրումը թույլատրելի չէ։ |
| False | Նկար-դաշտի խմբագրումը թույլատրելի է։  |

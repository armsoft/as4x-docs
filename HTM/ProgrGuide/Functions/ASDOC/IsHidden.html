Свойство IsHidden

См. также  Пример  Применяется к

Возвращает признак невидимости реквизита в описании документа.

Свойство для чтения.

 

Синтаксис

object.IsHidden(RekvName)  

Синтаксис свойства IsHidden состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
RekvName выражение типа Variant, определяющее идентификатор реквизита

 

Тип данных

Логический

# SetFocus մեթոդ

[См. также](../Asdoc.md) Օրինակ [Применяется к](../Asdoc.md)


Ֆոկուսը տեղափոխում է փաստաթղթի տրված դաշտի վրա։


## Շարահյուսություն

``` vb
object.SetFocus(RekvName)
```

Բաղադրիչներն են՝ 

| Պարամետր | Նկարագրություն |
|--|--|
| object | Փաստաթուղթ օբյեկտի հղում։|
| RekvName | Դաշտի ներքին անուն։ |

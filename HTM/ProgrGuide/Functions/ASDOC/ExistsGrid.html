Свойство ExistsGrid

См. также  Пример  Применяется к

Возвращает наличие грид-таблицы в описании документа.

Свойство для чтения.

 

Синтаксис

object.ExistsGrid(GridName)  

Синтаксис свойства ExistsGrid состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
GridName строковое выражение, определяющее идентификатор реквизита грид-таблицы

 

Тип данных

Логический

Свойство Parents

См. также  Пример  Применяется к

Возвращает коллекцию ISN-ов родительских документов для текущего. В системе несколько родительских документов могут породить один дочерний документ.

Свойство для чтения.

 

Синтаксис

object.Parents

Синтаксис свойства Parents состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Примечание

Отличие данного свойства от свойства ParentISN состоит в возврате ISN всех родителей, а не первого встретившегося.

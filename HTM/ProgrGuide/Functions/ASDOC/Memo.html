Свойство Memo

См. также  Пример  Применяется к

Возвращает или устанавливает значение для данного реквизита Memo.

Свойство для чтения и записи.

 

Синтаксис

object.Мемо (vMemoName)[=sValue]

Синтаксис свойства Memo состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа документ.
vMemoName строковое выражение, определяющее идентификатор реквизита Memo для данного документа.

# RefreshTS մեթոդ

[См. также](../Asdoc.md) Օրինակ [Применяется к](../Asdoc.md)

Հիշողության մեջ թարմացնում է փաստաթղթի վերջին գրանցման համարը տվյալների պահոցից։

## Շարահյուսություն

``` vb
object.RefreshTS
```
Բաղադրիչներն են՝ 

| Պարամետր | Նկարագրություն |
|--|--|
| object | Փաստաթուղթ օբյեկտի հղում։|

Метод StoreInTree

См. также  Пример  Применяется к

Сохраняет элемент дерева.

 

Синтаксис

object.StoreInTree ( xTreeEl )

Синтаксис метода StoreInTree состоит из следующих астей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
xTreeEl переменная, ссылающаяся на объект типа элемент дерева. Он должен быть заранее создан и инициализирован, потом только сохранен.

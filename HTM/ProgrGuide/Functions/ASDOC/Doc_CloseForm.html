Метод CloseForm

См. также  Пример  Применяется к

Закрывает форму документа.

 

Синтаксис

object.CloseForm

Синтаксис свойства CloseForm состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа.

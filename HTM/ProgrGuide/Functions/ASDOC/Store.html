Метод Store

См. также  Пример  Применяется к

Производит необходимые проверки и запоминает документ

 

Синтаксис

object.Store ([intCheckLevel])

Синтаксис метода Store состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа.
intCheckLevel необязательное численное выражение целого типа, определяющее уровень проверки документа.

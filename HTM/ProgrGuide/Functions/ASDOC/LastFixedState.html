Свойство LastFixedState

См. также  Пример  Применяется к

Возвращает последнее зафиксированное в базе состояние документа.

Свойство для чтения.

 

Синтаксис

object.LastFixedState

Синтаксис свойства LastFixedState состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа


Примечание

В отличие от свойства LastFixedState, свойство State возвращает текущее (возможно измененное в скрипте, но не запомненное) состояние документа.

 

Тип данных

Целое число

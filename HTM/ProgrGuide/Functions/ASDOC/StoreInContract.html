Метод StoreInContract

Сохраняет объект договора.

См. также Пример Применяется к

Синтаксис

object.StoreInContract(descContract)

Синтаксис свойства StoreInContract состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
descContract переменная ссылаяющаяся на объект типа AsContractDesc. Он должен быть заранее создан и инициализирован, потом только сохранен.

Примечание

Данный метод доступен только из скриптов банковского отдела.

Свойство DocType

См. также  Пример  Применяется к

Возвращает тип документа (т.е. его идентификатор).

Свойство для чтения.

 

Синтаксис

object.DocType

Синтаксис свойства DocType состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Строка

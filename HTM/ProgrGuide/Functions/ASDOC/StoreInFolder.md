# StoreInFolder մեթոդ


[См. также](../Functions/DocumentsCirculation/CreateFolderElement.html) Օրինակ [Применяется к](../Asdoc.md)

Գրանցում է թղթապանակի տարրը։

## Շարահյուսություն

``` vb
object.StoreInFolder ( xFoldEl )
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object | Փաստաթուղթ օբյեկտի հղում։ |
| xFoldEl | [Թղթապանակի տարրի հղում](../AsFoldElement.html)։ |

## Նկատառումներ
Այս մեթոդը հարկավոր է կանչել միմիայն փաստաթղթի Folders իրադարձության մշակիչի մեջ։ 


Թղթապանակի տարրի ստեղծումը կատարվում է [CreateFolderElement](../Functions/DocumentsCirculation/CreateFolderElement.html) ֆունկցիայի միջոցով։

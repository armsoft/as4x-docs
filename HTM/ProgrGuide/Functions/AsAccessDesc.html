Объект описатель доступа

Создание объекта описатель доступа обеспечивается через функцию AccessDesc.

Свойства Описание
Access Возвращает или устанавливает доступ пользователя.

 

Методы Описание
Clean Создает пустой дескриптор доступа для указанного объекта.
Load Загружает дескриптор доступа для указанного объекта.
Store Сохраняет дескриптор доступа объекта.

Свойство ElType

См. также  Пример  Применяется к

Возвращает имя внутреннего типа данных элемента документа.

Свойство только для чтения.

 

Синтаксис

object.ElType

Синтаксис свойства ElType состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент структуры документа.

 

Тип данных

String

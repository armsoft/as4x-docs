Свойство Attributes

См. также  Пример  Применяется к

Возвращает значение атрибута элемента документа.

Свойство только для чтения.

 

Синтаксис

object.Attributes

Синтаксис свойства Attributes состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент структуры документа.

 

Тип данных

Строка

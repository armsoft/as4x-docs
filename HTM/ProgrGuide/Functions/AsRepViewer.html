Объект отчет

Создание нового отчета в системе обеспечивается через функцию CreateRepViewer.

Свойства Описание
AllowHistory Возвращает или устанавливает разрешение на просмотр журнала Log документа из отчета.
AllowPrintPreview Возвращает или устанавливает разрешение на просмотр печатной формы документа из отчета.
AllowView Возвращает или устанавливает разрешение на просмотр документа из отчета.
AllowViewDocChildren Возвращает или устанавливает разрешение на просмотр дочерних документов из отчета.
AllowViewDocParents Возвращает или устанавливает разрешение на просмотр родительских документов из отчета.
Caption Возвращает или устанавливает заголовок отчета (справки).
CaptionSpreadsheetExportStyle Возвращает или устанавливает имя стиля экспорта загаловочной строки в Excel.
ECaption Возвращает или устанавливает заголовок отчета (справки) на иностранном языке.
DisableDocFunctions Устанавливает или снимает запрет на использование функций документа из отчета.
DocBased Возвращает или устанавливает признак oснованности отчета на документах.
ExportType Возвращает или устанавливает тип кодировки файла для экспортируемой справки.
FootCount Возвращает количество строк итогов (сносок) отчета.
Footer Возвращает строку итогов (сносок) по индексу.
FragCount Возвращает количество фрагментов в справке.
FragLength Возвращает длину заданного фрагмента справки.
GetRows Возвращает массив строк из справки, начиная с указанной строки с указанным количеством.
HasDefinedStyle Возвращает признак задания стиля экспорта строк справки в Excel.
HeadCount Возвращает количество строк заголовков отчета.
Header Возвращает указанную строку заголовка отчета.
InitialWidthInChars Возвращает или устанавливает ширину окна в символах при начальном показе.
MaxLength Возвращает сумму длин всех фрагментов справки.
PrintStyle Возвращает или устанавливает стиль печати для отчета.
ReportId Возвращает идентификатор отчета.
RowCount Возвращает количество строк в справке.
RowDescriptor Возвращает описатель данной строки.
RowISN Возвращает внутрисистемный уникальный идентификатор ISN для данной строки.
UnMoveFragment Возвращает или устанавливает наличие неподвижного фрагмента.
UnMoveHeadCount Возвращает или устанавливает количество неперемещаемых заголовков.
UseFormatting Возвращает или устанавливает признак использования форматирования текста в отчете.

 

Методы Описание
AddExportStyle Добавляет имя стиля экспорта данных строки в Excel.
AddFooter Добавляет в справке новую строку итогов.
AddFragment Добавляет в справке новый фрагмент заданной ширины.
AddHeader Добавляет в данной справке новую строку неподвижного заголовка.
AddRow Добавляет в данной справке новую строку.
Break В текущей строке справки вставляет символ прогона страницы.
LoadFrom Загружает описание и из тела отчета создает обьект отчет.
RegistrFunction Добавляет в контекстное меню вызов пользовательской функции.
RegistrNode Добавляет группировочный узел в подменю вызовов пользовательских функций. 
SaveAs Запоминает описание и тело отчета в файл.
SaveToFile Сохраняет отчет в файле.
Show Выводит подготовленную справку на экран.

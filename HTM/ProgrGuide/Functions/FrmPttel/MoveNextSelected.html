Метод MoveNextSelected

См. также  Пример  Применяется к

Перемещает указатель записи на следующую отмеченную строчку в текущей папке. При использовании данного метода в текущей папке с истинным значением свойства EOF генерируется ошибка. Если указатель находится на последней отмеченной строке и используется данный метод, то значение свойства EOF становится истинным.

 

Синтаксис

object.MoveNextSelected

Синтаксис метода MoveNextSelected состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

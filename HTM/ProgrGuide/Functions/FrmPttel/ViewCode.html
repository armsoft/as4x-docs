Свойство ViewCode

См. также  Пример  Применяется к

Возвращает идентификатор базового вида просмотра для текущей папки.

Свойство для чтения.

 

Синтаксис

object.ViewCode

Синтаксис свойства ViewCode состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

 

Тип данных

Строка

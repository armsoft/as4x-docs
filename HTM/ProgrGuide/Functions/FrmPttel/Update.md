## Շարահյուսություն

``` vb
object.Update( nISN )
```

Բաղադրիչներն են՝



| Պարամետր | Նկարագրություն |
|--|--|
| object | Ընթացիկ դիտելու ձևի հղում։  |
| nISN | Փաստաթղթի ներքին նույնականացման համարը։  |

# RowCount հատկություն

[См. также](../Frmpttel.md) Օրինակ [Применяется к](../Frmpttel.md)

Վերադարձնում է տողերի քանակը ընթացիկ թղթապանակում։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.RowCount
```
Բաղադրիչներն են՝


| Параметр | Описание |
|--|--|
| object | Ընթացիկ դիտելու ձևի հղում։  |

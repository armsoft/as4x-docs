Свойство Parameters

См. также  Пример  Применяется к

Возвращает значения параметров, использованных при вызове текущей папки.

Свойство для чтения.

 

Синтаксис

object.Parameters( index )

Синтаксис свойства Parameters состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.
index численное выражение целого типа, определяющее индекс параметра. Начальное значение индекса 1.

 

Тип данных

Варианта

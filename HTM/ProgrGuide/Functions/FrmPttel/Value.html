Свойство Value

См. также  Пример  Применяется к

Возвращает значение указанной перманентной колонки текущей строки в текущей папке. Если значения свойств EOF или BOF являются истинными, то использование данного свойства генерирует сообщение об ошибке.

Свойство для чтения.

 

Синтаксис

object.Value (sColName)

Синтаксис свойства Value состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.
sColName строковое выражение, определяющее идентификатор перманентной колонки в текущем виде просмотра

 

Примечание

Если в качестве аргумента передать идентификатор неперманентной колонки, то будет сгенерирована ошибка.

 

Тип данных

Варианта

Свойство SelectedCount

См. также  Пример  Применяется к

Возвращает количество отмеченных строк в текущей папке. Если отмеченных строк нет, то данное свойство все равно возвращает 0.

Свойство для чтения.

 

Синтаксис

object.SelectedCount

Синтаксис свойства SelectedCount состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

 

Тип данных

Длинное целое число

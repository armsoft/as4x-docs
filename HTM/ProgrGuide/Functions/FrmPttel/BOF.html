Свойство BOF

См. также  Пример  Применяется к

Проверяет выполнение условия "начало файла" для текущей папки. Свойство возвращает значение "истина" при попытке установки указателя записи перед первой строкой текущего открытого курсора данных или если предшествующие команды MovePrevious, MovePreviousSelected не были успешными.

Свойство для чтения.

 

Синтаксис

object.BOF

Синтаксис свойства BOF состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

 

Тип данных

Логический

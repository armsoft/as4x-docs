Свойство Selected

См. также  Пример  Применяется к

Возвращает или устанавливает признак отмеченности текущей строки в папке.

Свойство для чтения и записи.

 

Синтаксис

object.Selected = [bValue]

Синтаксис свойства Selected состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.
bValue логическое выражение, определяющее признак отмеченности строки

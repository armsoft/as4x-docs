Свойство RowCount

См. также  Пример  Применяется к

Возвращает количество строк в текущей папке.

Свойство для чтения.

 

Синтаксис

object.RowCount

Синтаксис свойства RowCount состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

 

Тип данных

Длинное целое

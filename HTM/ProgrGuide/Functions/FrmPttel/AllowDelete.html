Свойство AllowDelete

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на удаление документа из текущего вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.AllowDelete [=bValue] 

Синтаксис свойства AllowDelete состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта текущий вид просмотра
bValue логическое выражение, разрешающее удаление документа

 

Установки

Установки для bValue следующие:

Значение Описание
False Не разрешать удаление документа
True Разрешать удаление документа

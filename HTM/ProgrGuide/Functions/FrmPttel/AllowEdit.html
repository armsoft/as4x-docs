Свойство AllowEdit

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на корректировку документа из текущего вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.AllowEdit [=bValue] 

Синтаксис свойства AllowEdit состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта текущий вид просмотра
bValue логическое выражение, разрешающее корректировку документа

 

Установки

Установки для bValue следующие:

Значение Описание
False Не разрешать корректировку документа
True Разрешать корректировку документа

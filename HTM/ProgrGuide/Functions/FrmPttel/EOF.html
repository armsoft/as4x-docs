Свойство EOF

См. также  Пример  Применяется к

Проверяет выполнение условия "конец файла" для текущей папки. Свойство возвращает значение "истина" при попытке установки указателя записи после последней строки текущего открытого курсора данных или если последние команды MoveNext, MoveNextSelected не были успешными.

Свойство для чтения.

 

Синтаксис

object.EOF

Синтаксис свойства EOF состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

 

Тип данных

Логический

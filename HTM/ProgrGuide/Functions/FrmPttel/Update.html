Метод Update

См. также  Пример  Применяется к

Обновляет указанную строчку в текущей папке.

 

Синтаксис

object.Update( nISN )

Синтаксис метода Update состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.
nISN выражение типа Long, определяющее внутрисистемный идентификационный код ISN документа.

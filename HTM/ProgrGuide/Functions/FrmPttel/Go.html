Метод Go

См. также  Пример  Применяется к

Переводит указатель записи на строку с указанным значением ключа. Каждая строка в текущей папке имеет уникальный идентификатор (ключ), возвращаемый через свойство BookMark. При отсутствии строки с указанным ключевым значением генерируется сообщение об ошибке.

 

Синтаксис

object.Go ( nKey )

Синтаксис метода Go состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.
nKey выражение типа Long, определяющее значение ключа для строки перехода

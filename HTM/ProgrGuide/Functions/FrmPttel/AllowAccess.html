Свойство AllowAccess

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на установку доступов к документам из текущего вида просмотра.

Свойство для чтения и записи.

Синтаксис

object.AllowAccess[=bValue]

Синтаксис свойства AllowAccess состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта текущий вид просмотра
bValue логическое выражение, разрешающее установку доступа на документ

Установки

Установки для bValue следующие:

Значение Описание
False Не разрешать установку доступа на документ
True Разрешать установку доступа на документ

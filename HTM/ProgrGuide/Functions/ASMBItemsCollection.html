Объект коллекция строк списка

Каждый произвольный вспомогательный список может содержать коллекцию отобранных строк, если задан признак отбора нескольких строк. Элементами данной коллекции являются объекты типа строка списка.

Свойства Описание
Count Возвращает количество элементов в коллекции строк списка.
Item

Возвращает элемент из коллекции строк списка по ключу.
KeyList Возвращает или устанавливает список значений поисковой колонки для элементов коллекции.
LookUpColumn Возвращает или устанавливает имя той колонки, по которой производится поиск значения.

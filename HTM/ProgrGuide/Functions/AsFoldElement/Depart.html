Свойство Depart

См. также  Пример  Применяется к

Возвращает или устанавливает значение кода отдела элемента папки.

Свойство для чтения и записи.

 

Синтаксис

object.Depart= [sValue]

Синтаксис свойства Depart состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.
sValue строковое выражение, определяющее код отдела элемента в папке. Ее максимальная длина может быть 3 символa.

Свойство Status

См. также  Пример  Применяется к

Возвращает или устанавливает статус элемента в папке.

Свойство для чтения и записи.

 

Синтаксис

object.Status = [sValue]

Синтаксис свойства Status состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.
sValue строковое выражение, определяющее статус документа в папке.

 

Установки

Параметр sValue может быть сочетанием из следующих значений:

Значение Описание
"E" в папке находится оригинал документа
"C" значение по умолчанию, в папке находится копия документа
"S" в папке находится документ в режиме вторичного ввода
"V" в папке находится документ в режиме визирования

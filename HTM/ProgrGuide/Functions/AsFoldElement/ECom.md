# ECom հատկություն

[См. также](../AsFoldElement.md) Օրինակ [Применяется к](../AsFoldElement.md)

Վերադարձնում կամ նշանակում է թղթապանակի տարրի մեկնաբանությունը օտար լեզվով։


## Շարահյուսություն

``` vb
object.ECom = [sValue]
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Թղթապանակի տարրի հղում։ |
| sValue | Նոր արժեք։ |

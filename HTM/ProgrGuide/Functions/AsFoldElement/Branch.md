# Branch հատկություն

[См. также](../AsFoldElement.md) Օրինակ [Применяется к](../AsFoldElement.md)

Վերադարձնում կամ նշանակում է թղթապանակի գրասենյակի կոդը։

## Շարահյուսություն

``` vb
object.Branch = [sValue]
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Թղթապանակի տարրի հղում։ |
| sValue  | Նոր արժեք։ |

Свойство ECom

См. также  Пример  Применяется к

Возвращает или устанавливает комментарий элемента папки на иностранном языке.

Свойство для чтения и записи.

 

Синтаксис

object.ECom = [sValue]

Синтаксис свойства ECom состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.
sValue строковое выражение, определяющее значение комментария к элементу папки на иностранном языке.

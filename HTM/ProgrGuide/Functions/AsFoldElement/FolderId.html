Свойство FolderId

См. также  Пример  Применяется к

Возвращает идентификатор папки, в которую входит элемент.

Свойство для чтения.

 

Синтаксис

object.FolderId

Синтаксис свойства FolderId состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.

 

Тип данных

Строка

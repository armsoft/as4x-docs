# Depart հատկություն

[См. также](../AsFoldElement.md) Օրինակ [Применяется к](../AsFoldElement.md)

Վերադարձնում կամ նշանակում է թղթապանակի բաժնի կոդը։

## Շարահյուսություն

``` vb
object.Depart= [sValue]
```
Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Թղթապանակի տարրի հղում։ |
| sValue | Նոր արժեք։ |

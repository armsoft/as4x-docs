Свойство Key

См. также  Пример  Применяется к

Возвращает или устанавливает ключ элемента папки.

Свойство для чтения и записи.

 

Синтаксис

object.Key = [sValue]

Синтаксис свойства Key состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.
sValue строковое выражение, определяющее ключ элемента в папке

 

Примечание

Если данное свойство не задано, то в качестве ключа по умолчанию берется ISN документа для элемента папки.

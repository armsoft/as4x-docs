Свойство Spec

См. также  Пример  Применяется к

Возвращает или устанавливает специальную строку элемента папки. Служит для хранения дополнительной информации об экземпляре документа, хранящемся в папке.

Свойство для чтения и записи.

 

Синтаксис

object.Spec = [sValue]

Синтаксис свойства Spec состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.
sValue строковое выражение, определяющее специальную строку элемента в папке. Ее максимальная длина может быть 255 символов.

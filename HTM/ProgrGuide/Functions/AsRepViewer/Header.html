Свойство Header

См. также  Пример  Применяется к

Возвращает указанную строку заголовка отчета.

Свойство для чтения.

 

Синтаксис

object.Header(index)

Синтаксис свойства Header состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
index численное выражение, определяющее номер строки заголовка отчета

 

Тип данных

Строка

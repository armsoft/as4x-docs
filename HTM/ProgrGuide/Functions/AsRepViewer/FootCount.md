# FootCount հատկություն

[См. также](../AsRepViewer.md) Օրինակ [Применяется к](../AsRepViewer.md) 

Վերադարձնում է տեղեկատու-հաշվետվության հատվածների քանակը։

## Շարահյուսություն

``` vb
object.FootCount
```

Բաղադրիչներն են՝ 


| Պարամետր | Նկարագրություն |
|--|--|
| object | Հաշվետվության հղում։  |

Свойство CaptionSpreadsheetExportStyle

См. также  Пример  Применяется к

Возвращает или устанавливает имя стиля экспорта заголовочной строки справки в Excel, заданного с помощью метода AddExportStyle.

Свойство для чтения и записи.

 

Синтаксис

object.CaptionSpreadsheetExportStyle[ = value ]

Синтаксис свойства CaptionSpreadsheetExportStyle состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
value строковое выражение, определяющее имя стиля экспорта заголовочной строки справки в Excel, заданного с помощью метода AddExportStyle.

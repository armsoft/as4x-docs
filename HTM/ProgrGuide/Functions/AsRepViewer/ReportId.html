Свойство ReportId

См. также  Пример  Применяется к

Возвращает идентификатор отчета.

Свойство для чтения.

 

Синтаксис

object.ReportId

Синтаксис свойства ReportId состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.

 

Тип данных

Строка

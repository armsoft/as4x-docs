Свойство FragLength

См. также  Пример  Применяется к

Возвращает длину заданного фрагмента справки.

Свойство для чтения.

 

Синтаксис

object.FragLength(index)

Синтаксис свойства FragLength состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
index числовое выражение, определяющее номер фрагмента

 

Тип данных

Целое число

Свойство AllowPrintPreview

 См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на просмотр печатной формы документа из отчета.

Свойство для чтения и записи.

 

Синтаксис

object.AllowPrintPreview[ = bValue ]

Синтаксис свойства AllowPrintPreview состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
bValue логическое выражение, разрешающее просмотр печатной формы документа.

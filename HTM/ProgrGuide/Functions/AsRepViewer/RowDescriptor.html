Свойство RowDescriptor

См. также  Пример  Применяется к

Возвращает описатель данной строки.

Свойство для чтения.

 

Синтаксис

object.RowDescriptor (lRow)

Синтаксис свойства RowDescriptor состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
lRow числовое выражение, определяющее номер строки.

 

Тип данных

Строковое выражение

Метод AddExportStyle

См. также  Пример  Применяется к

Задает имя стиля экспорта данных строки в Excel на основе уже существующих описаний, заданных с помощью метода AddPartt объекта типа SpreadsheetExportRowStyle.

 

Синтаксис

object.AddExportStyle(styleName, sreadsheetExportStyle)

Синтаксис метода AddExportStyle состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
sStyleNamestyleName строковое выражение, определяющее имя стиля экспорта данных строки в Excel.
spreadsheetExportStylee строковое выражение, определяющее ссылку на экземпляр объекта типа SpreadsheetExportRowStyle.

# Header հատկություն

[См. также](../AsRepViewer.html) Օրինակ [Применяется к](../AsRepViewer.html) 

Վերադարձնում է հաշվետվության գլխագրի տողը ըստ ինդեքի։

Կարդալո հատկություն։

## Շարահյուսություն 

``` vb
object.Header(index)
```


Բաղադրիչներն են՝ 


| Պարամետր | Նկարագրություն |
|--|--|
| object | Հաշվետվության հղում։  |
| index | Գլխագրի տողի համարը։  |

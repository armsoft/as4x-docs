Свойство RowCount

См. также  Пример  Применяется к

Возвращает количество строк в справке.

Свойство для чтения.

 

Синтаксис

object.RowCount

Синтаксис свойства RowCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.

 

Тип данных

Длинное целое число

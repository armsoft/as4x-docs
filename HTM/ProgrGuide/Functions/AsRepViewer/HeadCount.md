# HeadCount հատկություն

[См. также](../AsRepViewer.md) Օրինակ [Применяется к](../AsRepViewer.md) 

Վերադարձնում է հաշվետվության գլխագիր տողերի քանակը։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.HeadCount
```

Բաղադրիչներն են՝ 


| Պարամետր | Նկարագրություն |
|--|--|
| object | Հաշվետվության հղում։  |

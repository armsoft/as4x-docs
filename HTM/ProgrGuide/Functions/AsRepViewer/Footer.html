Свойство Footer

См. также  Пример  Применяется к

Возвращает строку итогов (сносок) по индексу.

Свойство для чтения.

 

Синтаксис

object.Footer(index)

Синтаксис свойства Footer состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
index численное выражение, определяющее номер строки итога.

 

Тип данных

Строка

Свойство FootCount

См. также  Пример  Применяется к

Возвращает количество строк итогов (сносок) отчета.

Свойство для чтения.

 

Синтаксис

object.FootCount

Синтаксис свойства FootCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.

 

Тип данных

Целое число

Свойство InitialWidthInChars

См. также  Пример  Применяется к

Возвращает или устанавливает ширину окна в символах при начальном показе. Если не задан, ширина расчитывается исходя из длин фрагментов.

Свойство для чтения и записи.

 

Синтаксис

object.InitialWidthInChars = [nValue]

Синтаксис свойства InitialWidthInChars состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
nValue численное выражение целого типа, определяющее ширину окна в символах при начальном показе.

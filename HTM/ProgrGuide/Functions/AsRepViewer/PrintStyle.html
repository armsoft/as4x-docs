Свойство PrintStyle

 См. также  Пример  Применяется к

Возвращает или устанавливает стиль печати для отчета. Заранее описанный и загруженный стиль печати с помощью данного свойства приписывается к отчету, для определения значений печатных параметров по умолчанию.

Свойство для чтения и записи.

 

Синтаксис

object.PrintStyle[ = sValue ]

Синтаксис свойства PrintStyle состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
sValue строковое выражение, определяющее идентификатор стиля печати

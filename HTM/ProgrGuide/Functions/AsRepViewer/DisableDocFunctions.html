Свойство DisableDocFunctions

См. также  Пример  Применяется к

Устанавливает или снимает запрет на использование функций документа из отчета.

Свойство для чтения и записи.

 

Синтаксис

object.DisableDocFunctions [=bValue] 

Синтаксис свойства DisableDocFunctions состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
bValue логическое выражение, запрещающее использование функций документа


Установки

Установки для bValue следующие:

Значение Описание
False Разрешать использование функций документа
True Запретить использование функций документа

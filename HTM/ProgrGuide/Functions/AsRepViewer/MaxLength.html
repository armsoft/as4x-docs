Свойство MaxLength

См. также  Пример  Применяется к

Возвращает сумму длин всех фрагментов справки.

Свойство для чтения.

 

Синтаксис

object.MaxLength

Синтаксис свойства MaxLength состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.

 

Тип данных

Целое число

#  Footer հատկություն

[См. также](../AsRepViewer.md) Օրինակ [Применяется к](../AsRepViewer.md) 

Վերադարձնում է հաշվետվության  տողատակի տողը ըստ ինդեքի։ 

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.Footer(index)
```

Բաղադրիչներն են՝ 


| Պարամետր | Նկարագրություն |
|--|--|
| object | Հաշվետվության հղում։  |
| index | Տողի համարը։  |

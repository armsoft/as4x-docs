Свойство RowISN

См. также  Пример  Применяется к

Возвращает внутрисистемный уникальный идентификатор ISN для данной строки.

Свойство для чтения.

 

Синтаксис

object.RowISN (lRow)

Синтаксис свойства RowISN состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
lRow числовое выражение, определяющее номер строки.

 

Тип данных

Длинное целое число

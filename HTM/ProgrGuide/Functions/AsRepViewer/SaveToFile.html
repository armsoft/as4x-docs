Метод SaveToFile

См. также  Пример  Применяется к

Сохраняет отчет в файле. Можно также задать режим кодировки.

 

Синтаксис

object.SaveToFile sFileName, [nMode], [bReplaceTag]

Синтаксис метода SaveToFile состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
sFileName строковое выражение, определяющее путь к файлу.
nMode необязательное численное выражение типа Integer, определяющее режим кодировки файла. При значении 1 - Windows ANSII, а при 2 - OEM DOS. По умолчанию принимает значение 1.
bReplaceTag необязательное логическое выражение, определяющее признак удаления тегов из текста(см. описание возможных тегов в UseFormatting).

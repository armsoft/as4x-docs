Метод DocBased

См. также  Пример  Применяется к

Возвращает или устанавливает признак oснованности отчета на документах.

Свойство для чтения и записи.

 

Синтаксис

object.DocBased [=bValue] 

Синтаксис свойства DocBased состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта отчет.
bValue логическое выражение, определяющее oснованность отчета на документах.

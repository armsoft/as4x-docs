Свойство UnMoveFragment

См. также  Пример  Применяется к

Возвращает или устанавливает наличие неподвижного фрагмента.

Свойство для чтения и записи.

 

Синтаксис

object.UnMoveFragment = [Value]

Синтаксис свойства UnMoveFragment состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
Value логическое выражение, определяющее наличие фиксированного, неподвижного фрагмента

 

Примечание

Фиксированным неподвижным фрагментом считается всегда первый фрагмент среди описанных.

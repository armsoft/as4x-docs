Свойство FragCount

См. также  Пример  Применяется к

Возвращает количество фрагментов в справке.

Свойство для чтения.

 

Синтаксис

object.FragCount

Синтаксис свойства FragCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.

 

Тип данных

Целое число

# TreeId հատկություն

[См. также](../AsTreeElement.md) Օրինակ [Применяется к](../AsTreeElement.md)

ծառի տարրի համար վերադարձնում է ծառի ներքին անունը։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.TreeId
```
Բաղադրիչներն են՝ 

| Պարամետր | Նկարագրություն |
|--|--|
| object | Ծառի տարրի հղում։ |

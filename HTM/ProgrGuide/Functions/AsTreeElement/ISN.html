Свойство ISN

См. также  Пример  Применяется к

Возвращает внутрисистемный идентификационный код ISN документа элемента дерева.

Свойство для чтения.

 

Синтаксис

object.ISN

Синтаксис свойства ISN состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент дерева.

 

Тип данных

Целое число

# Key հատկություն

[См. также](../AsTreeElement.html) Օրինակ [Применяется к](../AsTreeElement.md)

Վերադարձնում կամ նշանակում է ծառի բանալին (կոդը)։


## Շարահյուսություն

``` vb
object.Key = [sValue]
```

Բաղադրիչներն են՝ 


| Պարամետր |Նկարագրություն |
|--|--|
| object | Ծառի տարրի հղում։ |
| sValue | Ծառի բանալին։ |

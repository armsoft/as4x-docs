Свойство Com

См. также  Пример  Применяется к

Возвращает или устанавливает комментарий элемента дерева на родном языке. Комментарий появляется в списке в виде наименования, во время ссылок на справочники типа Tree.

Свойство для чтения и записи.

 

Синтаксис

object.Com = [sValue]

Синтаксис свойства Com состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент дерева.
sValue строковое выражение, определяющее значение комментария к элементу дерева.

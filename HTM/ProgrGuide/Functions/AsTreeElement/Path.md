# Path հատկություն

[См. также](../AsTreeElement.md) Օրինակ [Применяется к](../AsTreeElement.md)

Ծառի մեջ վերադարձնում է տարրի ճանապարհը, սկսելով ծառի արմատից։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.Path
```
Բաղադրիչներն են՝


| Պարամետր | ՆկարագրությունՍ |
|--|--|
| object| Ծառի տարրի հղում։ |

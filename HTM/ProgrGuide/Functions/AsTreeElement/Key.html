Свойство Key

См. также  Пример  Применяется к

Возвращает или устанавливает ключ (код) элемента дерева.

Свойство для чтения и записи.

 

Синтаксис

object.Key = [sValue]

Синтаксис свойства Key состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент дерева.
sValue строковое выражение, определяющее ключ элемента в дереве

Свойство Leaf

См. также  Пример  Применяется к

Возвращает признак лепестка в дереве.

Свойство для чтения.

 

Синтаксис

object.Leaf

Синтаксис свойства Leaf состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент дерева.

 

Тип данных

Возвращает "1" - если элемент дерева является лепестком последнего уровня,  "0" - если элемент является узлом и сам имеет лепестки.

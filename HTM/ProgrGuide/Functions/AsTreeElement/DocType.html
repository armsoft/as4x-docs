Свойство DocType

См. также  Пример  Применяется к

Возвращает тип документа для элемента дерева.

Свойство для чтения.

 

Синтаксис

object.DocType

Синтаксис свойства DocType состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент дерева.

 

Тип данных

Строка

Свойство Color

См. также  Пример  Применяется к

Возвращает или устанавливает цвет шрифта ячейки.

 

Синтаксис

object.Color[= value]

Синтаксис свойства Color состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа TemplateSubstCellStyle.
value строковое выражение, oпределяющее цвет шрифта ячейки, которое начинается с символа # и представляется в виде трех пар цифер шестнадцатеричной системы цветовой модели  RGB.

Свойство Italic

См. также  Пример  Применяется к

Возвращает или устанавливает курсивное начертание шрифта ячейки.

 

Синтаксис

object.Italic[= value]

Синтаксис свойства Italic состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа TemplateSubstCellStyle.
value численное выражение, oпределяющее курсивное начертание шрифта ячейки. Принимает значения 0, 1 или 2. При значении 1 устанавливается курсивное начертание шрифта ячейки, при значении 0 курсивное начертание шрифта снимается, при значении 2 ничего не меняется. По умолчанию принимает значение 2.

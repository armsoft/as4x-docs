Свойство FullFileName

См. также  Пример  Применяется к

Возвращает полный путь файлового дополнения на данном компьютере.

Свойство только для чтения.

 

Синтаксис

object.FileName

Синтаксис свойства FileName состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта файловое дополнение.

 

Тип данных

Строка

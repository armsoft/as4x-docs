# Comment հատկությւն

[См. также](../AsAttachment.md) Օրինակ [Применяется к](../AsAttachment.md) 

Վերադարձնում կամ նշանակում է  կցված ֆայլի մեկնաբանությունը։

## Շարահյուսություն

``` vb
object.Comment [=sValue]
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրոթյոն |
|--|--|
| object | Փաստաթղթի կցված ֆայլի հղում։ |
| sValue | Նոր արժեք։  |

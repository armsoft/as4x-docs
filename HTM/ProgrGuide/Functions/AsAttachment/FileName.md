# FileName հատկություն

[См. также](FullFileName.md) Օրինակ [Применяется к](../AsAttachment.md)

Վերադարձնում է  կցված ֆայլի անվանումը։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.FileName
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրոթյուն |
|--|--|
| object | Փաստաթղթի կցված ֆայլի հղում։ |

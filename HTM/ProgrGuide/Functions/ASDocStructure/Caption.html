Свойство Caption

См. также  Пример  Применяется к

Возвращает или устанавливает заголовок документа.

Синтаксис

object.Caption

Синтаксис свойства Caption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта структура документа.

 

Тип данных

Строка

Свойство Element

 См. также  Пример  Применяется к

Возвращает элемент структуры документа  первого уровня.

Свойство только для чтения. Свойство по умолчанию.

 

Синтаксис

object.Element (vNameMBItem)

Синтаксис свойства Element состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта структура документа.
vNameMBItem выражение типа Variant, определяющее ключ элемента в коллекции структурных элементов документа. Ключом элемента может быть или порядковый номер элемента в коллекции, или идентификатор элемента (идентификатор реквизита, идентификатор грид-таблицы, и т.д.).

 

Тип данных

Элемент структуры документа.

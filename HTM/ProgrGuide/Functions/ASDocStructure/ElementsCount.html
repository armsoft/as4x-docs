Свойство ElementsCount

См. также  Пример  Применяется к

Возвращает количество структурных элементов первого уровня элементов в документе.

Свойство только для чтения.

 

Синтаксис

object.ElementsCount

Синтаксис свойства ElemetsCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта структура документа.

 

Тип данных

Целое число

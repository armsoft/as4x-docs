Свойство Caption

См. также  Пример  Применяется к

Возвращает заголовок учета.

Свойство для чтения.

 

Синтаксис

object.Caption

Синтаксис свойства Caption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа учет

 

Тип данных

Строка

Свойство AccFolder

См. также  Пример  Применяется к

Возвращает идентификатор папки счетов из описания учетов.

Свойство для чтения.

 

Синтаксис

object.AccFolder

Синтаксис свойства AccFolder состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа учет

 

Примечание

Свойство возвращает идентификатор папки, которая содержит участвующие в дебетовании или кредитовании счета. А объект проводки должен находится в папке объектов учета MainFolder.

Свойство Description

См. также  Пример  Применяется к

Возвращает краткий разъяснительный текст учета из его описания.

Свойство для чтения.

 

Синтаксис

object.Description

Синтаксис свойства Description состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа учет

 

Тип данных

Строка

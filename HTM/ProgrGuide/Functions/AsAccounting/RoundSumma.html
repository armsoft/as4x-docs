Свойство RoundSumma

См. также  Пример  Применяется к

Возвращает степень округления суммы в НДЕ для проводок данного учета. При сохранении проводок методом StoreFact, сумма проводки в основной валюте округляется до заданной позиции после десятичной запятой.

Свойство для чтения.

 

Синтаксис

object.RoundSumma

Синтаксис свойства RoundSumma состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа учет

 

Тип данных

Длинное целое

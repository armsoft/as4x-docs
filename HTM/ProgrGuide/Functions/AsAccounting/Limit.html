Свойство Limit

См. также  Пример  Применяется к

Возвращает признак автоматической проверки лимитов по объектам учета.

Свойство для чтения.

 

Синтаксис

object.Limit

Синтаксис свойства Limit состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа учет

 

Тип данных

Логический

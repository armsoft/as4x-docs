# Show մեթոդ

[См. также](../AsDialog.md) [Օրինակ](../../Examples/E_AsDialog.html) [Применяется к](../AsDialog.md)

Ակտիվացնում է երկխոսության պատուհանը։

## Շարահյուսություն

``` vb
object.Show
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object | Երկխոսության պատուհանի հղում։ |

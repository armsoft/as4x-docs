# Name հատկություն

[См. также](../AsDocStructureElement.md) Օրինակ [Применяется к](../AsDocStructureElement.md)

Վերադարձնում է տարրի ներքին անունը։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.Name
```

Բաղադրիչներն են՝ 

| Պարամետր | Նկարագրություն |
|--|--|
| object  | Փաստաթղթի տարրի կառուցվածքի հղում։ |

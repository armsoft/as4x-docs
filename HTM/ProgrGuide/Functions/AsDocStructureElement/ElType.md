# ElType հատկություն

[См. также](../../types.html) Օրինակ [Применяется к](../AsDocStructureElement.md) 

Վերադարձնում է տարրի ներքին տեսակը (C, SUMMA, NP...)։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.ElType
```

Բաղադրիչներն են՝ 

| Պարամետր | Նկարագրություն |
|--|--|
| object  | Փաստաթղթի տարրի կառուցվածքի հղում։ |

Объект транспонирование данных

Создание объекта транспонирования данных обеспечивается через функцию CreatePivotInfo.

 

Свойства Описание
PivotColumns Возвращает транспонированные колонки.
PivotData Возвращает транспонированные данные.
PivotRows Возвращает транспонированные строки.

 

Методы Описание
AddPivotColumn Транспонирует заданную колонку.
AddPivotData Транспонирует  данные заданной колонки.
AddPivotRow Транспонирует  строки заданной колонки.

Свойство Value

См. также  Пример  Применяется к

Возвращает или устанавливает значение элемента управления диалога.

Свойство для чтения и записи. Данное свойство является свойством по умолчанию, т.е. при использовании его можно опустить.

 

Синтаксис

object.Value(sControlName) = [vValue]

Синтаксис свойства Value состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.
sControlName строковое выражение, определяющее идентификатор элемента управления в описательном диалоге.
vValue выражение типа Variant, определяющее значение злемента управления в описательном диалоге.

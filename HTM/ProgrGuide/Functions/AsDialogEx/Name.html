Свойство Name

См. также  Пример  Применяется к

Возвращает идентификатор диалога.

Свойство для чтения.

 

Синтаксис

object.Name

Синтаксис свойства Name состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.

 

Тип данных

Строка

Свойство AddAdditionalRegKey

См. также  Пример  Применяется к

Возвращает или устанавливает дополнительное расширение для запоминания и загрузки значений из реестра.

Свойство для чтения и записи.

 

Синтаксис

object.AddAdditionalRegKey = [sValue]

Синтаксис свойства AddAdditionalRegKey состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.
sValue строковое выражение, определяющее расширение для запоминания и загрузки значений из реестра.

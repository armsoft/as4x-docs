Свойство Control

См. также  Пример  Применяется к

Возвращает элемент управления описательного диалога с данным именем.

Свойство для чтения. Возвращает значение типа объект.

 

Синтаксис

object.Control(sControlName)

Синтаксис свойства Control состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.
sControlName строковое выражение, определяющее имя элемента управления в описательном диалоге.

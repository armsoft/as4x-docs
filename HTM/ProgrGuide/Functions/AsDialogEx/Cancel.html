Свойство Cancel

См. также  Пример  Применяется к

Возвращает или устанавливает тип выхода из описательного диалога.

Свойство для чтения и записи.

 

Синтаксис

object.Cancel = [bValue]

Синтаксис свойства Cancel состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.
bValue выражение логического типа, определяющее тип выхода из диалога, с событием Cancel, т.е. кнопка OK не нажата.

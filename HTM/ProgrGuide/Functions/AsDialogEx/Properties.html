Свойство Properties

См. также  Пример  Применяется к

Возвращает временную коллекцию Dictionary для пользовательской обработки.

Свойство для чтения.

 

Синтаксис

object.Properties

Синтаксис свойства Properties состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.

 

Тип данных

Объект

Свойство AllowView

См. также  Пример  Применяется к

Устанавливает разрешение на просмотр документа из шаблона печати.

Свойство для записи.

 

Синтаксис

object.AllowView[ = bValue ]

Синтаксис свойства AllowView состоит из следующих частей:

Параметр Описание
object строковое выражение, ссылающееся на объект печатной формы.
bValue логическое выражение, разрешающее просмотр документа.

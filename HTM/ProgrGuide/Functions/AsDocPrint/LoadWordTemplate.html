Метод LoadWordTemplate

См. также  Пример  Применяется к

Загружает шаблонную форму печати документа в формате Word и создает ссылку на файл Word-а.

 

Синтаксис

object.LoadWordTemplate (sTmplName)

Синтаксис метода LoadWordTemplate состоит из следующих частей:

Параметр Описание
object строковое выражение, ссылающееся на объект печатной формы.
sTmplName строковое выражение, определяющее идентификатор шаблона

 

Примечание

Данный метод вызывается из обработчиков системных событий Print, PrintExcel, PrintWord.

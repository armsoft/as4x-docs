Свойство Node

См. также  Пример  Применяется к

Возвращает указанный элемент дерева в виде коллекции.

Свойство для чтения.

 

Синтаксис

object.Node(index).Item(ItemKey)

Синтаксис свойства Node состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на форму прокрутки деревьев-справочников
index численное выражение типа Long, определяющее номер строки в форме прокрутки деревьев. Индекс начинается с 0 до NodesCount-1.
ItemKey строковое выражение, определяющее идентификатор ключа в коллекции значений

 

Установки

Установки для ItemKey следующие:

Значение Описание
CODE код выбранного узла
NAME наименование выбранного узла
ISN внутрисистемный идентификационный код для выбранного узла
PARENTCODE код непосредственного родителя выбранного узла
PARENTNAME наименование непосредственного родителя выбранного узла
PARENTISN внутрисистемный идентификационный код непосредственного родителя для выбранного узла

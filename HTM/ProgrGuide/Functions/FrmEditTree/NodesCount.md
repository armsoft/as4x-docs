# NodesCount հատկություն

[См. также](../FrmEditTree.md) Օրինակ [Применяется к](../FrmEditTree.md)

Ընթացիկ ծառում վերադարձնում է հանգույցների քանակը։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.NodesCount
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object |  Ընթացիկ ծառի հղում։|

Свойство CurrentNode

См. также  Пример  Применяется к

Возвращает текущий элемент дерева в виде коллекции значений.

Свойство для чтения.

 

Синтаксис

object.CurrentNode.Item(item)

Свойство возвращает значение в виде коллекции.

Синтаксис свойства CurrentNode состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на форму прокрутки деревьев-справочников
item строковое выражение, определяющее ключ в коллекции

 

Установки

Установки для item следующие:

Значение Описание
CODE код выбранного узла
NAME наименование выбранного узла
ENAME иностранное наименование выбранного узла
ISN внутрисистемный идентификационный код для выбранного узла
PARENTCODE код непосредственного родителя выбранного узла. Для одноуровневого дерева код родителя пустой.
PARENTNAME наименование непосредственного родителя выбранного узла
PARENTISN внутрисистемный идентификационный код непосредственного родителя для выбранного узла

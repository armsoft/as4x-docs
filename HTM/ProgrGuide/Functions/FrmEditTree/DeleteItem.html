Метод DeleteItem

См. также  Пример  Применяется к

Удаляет текущий лепесток из дерева. Если лепесток дерева является документным, то документ также будет удален.

 

Синтаксис

object.DeleteItem

Синтаксис метода DeleteItem состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на форму прокрутки деревьев-справочников

 

Примечание

Перед вызовом данного метода необходимо сделать все необходимые проверки на корректность.

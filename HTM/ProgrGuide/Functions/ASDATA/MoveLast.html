Метод MoveLast

См. также  Пример  Применяется к

Передвигает позицию маркера в открытом источнике данных на последнюю строку. При перемещениях маркера следует учитывать тип открытого источника данных.

 

Синтаксис

object.MoveLast

Синтаксис метода MoveLast состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

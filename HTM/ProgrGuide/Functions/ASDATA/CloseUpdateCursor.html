Метод CloseUpdateCursor

См. также  Пример  Применяется к

Закрывает открытый курсор обновления источника данных.

 

Синтаксис

object.CloseUpdateCursor

Синтаксис метода CloseUpdateCursor состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

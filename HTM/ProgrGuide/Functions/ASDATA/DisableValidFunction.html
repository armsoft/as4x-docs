Свойство DisableValidFunction

См. также  Пример  Применяется к

Свойство позволяет блокировать вызов события Valid источника данных. Обычно значение True применяется для повышение производителности, когда по значениям параметров можно определить что вызов события Valid не нужен.

Синтаксис

object.DisableValidFunction

Синтаксис свойства DisableValidFunction состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

 

Тип данных

Логическое

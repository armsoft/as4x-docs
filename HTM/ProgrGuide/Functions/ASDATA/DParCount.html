Свойство DParCount

См. также  Пример  Применяется к

Возвращает количество параметров к SQL запросу в источнике данных.

Свойство только для чтения.

 

Синтаксис

object.DParCount

Синтаксис свойства DParCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

 

Тип данных

Целое число

Метод CloseCursor

См. также  Пример  Применяется к

Закрывает открытый источник данных.

 

Синтаксис

object.CloseCursor

Синтаксис метода CloseCursor состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

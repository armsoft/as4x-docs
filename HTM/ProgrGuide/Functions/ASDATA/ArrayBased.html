Свойство ArrayBased

См. также  Пример  Применяется к

Возвращает признак того, является ли источник данных основанным на SQL запросе (ArrayBased=0) или на объекте типа XArrayDB (ArrayBased=1). Если значение выражения равно 1, то обязательно должно быть обработано событие Load.

Свойство только для чтения.

 

Синтаксис

object.ArrayBased

Синтаксис свойства ArrayBased состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

 

Тип данных

Логическое

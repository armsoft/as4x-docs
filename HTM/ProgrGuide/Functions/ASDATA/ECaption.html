Свойство ECaption

См. также  Пример  Применяется к

Возвращает заголовок источника данных на иностранном языке.

Свойство только для чтения.

 

Синтаксис

object.ECaption

Синтаксис свойства ECaption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

 

Тип данных

Строка

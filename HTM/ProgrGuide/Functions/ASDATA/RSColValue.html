Функция RSColValue

См. также  Пример  Применяется к

Возвращает значение колонки результата SQL запроса на котором основан источник данных. Вызов функции возможен для иточников данных основанных на SQL запросе.

 

Синтаксис

object.RSColValue(sColName)

Синтаксис свойства RSColValue состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.
sColName строковое выражение типа, определяющее идентификатор колонки в результате  SQL  запроса.

 

Тип данных

Варианта

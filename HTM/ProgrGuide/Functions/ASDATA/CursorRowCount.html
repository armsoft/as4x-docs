Свойство CursorRowCount

См. также  Пример  Применяется к

Возвращает количество строк в открытом источнике данных, если он открыт опцией ASOpenKeyset. Если источник данных открыт любой другой опцией, то свойство возвращает значение -1.

Свойство только для чтения.

 

Синтаксис

object.CursorRowCount

Синтаксис свойства CursorRowCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

 

Тип данных

Целое число

Свойство Formatted

См. также  Пример  Применяется к

Возвращает символьное значение колонки для текущей строки курсора источника данных.

Свойство для чтения.

 

Синтаксис

object.Formatted(sColName)

Синтаксис свойства Formatted состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.
sColName выражение типа Variant, определяющее идентификатор колонки в источнике данных.

 

Примечание

Использование данной функции удобно для получения строк фиксированных длин при формировании отчетов. Для получения строки учитывается тип колонки источника данных.

 

Тип данных

Строка

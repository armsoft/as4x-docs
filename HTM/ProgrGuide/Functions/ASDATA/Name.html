Свойство Name

См. также  Пример  Применяется к

Возвращает идентификатор источника данных.

Свойство только для чтения.

 

Синтаксис

object.Name

Синтаксис свойства Name состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

 

Тип данных

Строка

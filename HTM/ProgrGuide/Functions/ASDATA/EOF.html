Свойство EOF

См. также  Пример  Применяется к

Возвращает признак конца в источнике данных.

Свойство для чтения.

 

Синтаксис

object.EOF

Синтаксис свойства EOF состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

 

Тип данных

Логический

Метод OpenUpdateCursor

См. также  Пример  Применяется к

Открывает курсор обновления источника данных. Курсор - это логическое множество строк в источнике данных.

 

Синтаксис

object.OpenUpdateCursor(lngISN)

Синтаксис метода OpenUpdateCursor состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.
lngISN численное выражение типа Long, определяющее ISN документа для которого происходит обновление.

Свойство CurrentIndicate

См. также  Пример  Применяется к

Возвращает текущий вид индикатора выполнения запроса источника данных.

Свойство только для чтения.

 

Синтаксис

object.CurrentIndicate

Синтаксис свойства CurrentIndicate состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

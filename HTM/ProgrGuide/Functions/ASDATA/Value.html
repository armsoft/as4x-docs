Свойство Value

См. также  Пример  Применяется к

Возвращает значение колонки источника данных для текущей строки. Данное свойство является свойством по умолчанию, что означает возможность опускания ключевого слова Value при его использовании.

Свойство для чтения.

 

Синтаксис

object.Value(sColName)

Синтаксис свойства Value состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.
sColName строковое выражение типа, определяющее идентификатор колонки в источнике данных.

 

Примечание

Обращения xDS.Value("STARTREM") и xDS("STARTREM") являются эквивалентными.

Метод MovePrevious

См. также  Пример  Применяется к

Передвигает позицию маркера в открытом источнике данных на предыдущую строку. При перемещениях маркера следует учитывать тип открытого источника данных.

 

Синтаксисс

object.MovePrevious

Синтаксис метода MovePrevious состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.

Объект произвольный вспомогательный список

Создание экземпляра объекта типа произвольный вспомогательный список производится через объявление переменной:
    Dim VarName as New AsModalBrowser

Свойства Описание
Height Возвращает или устанавливает высоту окна произвольного вспомогательного списка.
isSelected Возвращает признак того, выбрана ли строка из произвольного вспомогательного списка.
Left Возвращает или устанавливает удаленность левого края произвольного вспомогательного списка от левого края окна  приложения.
LookUpColumn Возвращает или устанавливает имя той колонки, по которой производится поиск значения для установки начальной позиции курсора при активизации произвольного вспомогательного списка.
LookUpSoftMode Возвращает или устанавливает тип поиска значения для установки начальной позиции курсора при активизации произвольного вспомогательного списка.
LookUpValue Возвращает или устанавливает значение в колонке, на строку которой устанавливается курсор при активизации произвольного вспомогательного списка.
MultiSelect

Возвращает или устанавливает признак того, может ли выборка произвольного вспомогательного списка содержать несколько строк.
SelectedItems Возвращает коллекцию отмеченных строк произвольного вспомогательного списка.
StartUpAtCentre Возвращает или устанавливает признак показа произвольного вспомогательного списка в центре экрана.
Top Возвращает или устанавливает удаленность верхнего края произвольного вспомогательного списка от верхнего края окна приложения.
Value Возвращает значение выбранной колонки в произвольном вспомогательном списке.
Width Возвращает или устанавливает ширину окна произвольного вспомогательного списка.

 

Методы Описание
Show Активизирует, выводя на экран, произвольный вспомогательный список.

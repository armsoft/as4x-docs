Свойство AllowDelete

См. также  Пример  Применяется к

Возвращает разрешение на удаление документа из дерева.

Свойство только для чтения.

 

Синтаксис

object.AllowDelete

Синтаксис свойства AllowDelete состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево

 

Тип данных

Логический

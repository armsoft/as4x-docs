Свойство TreeId

См. также  Пример  Применяется к

Возвращает значение идентификатора дерева.

Свойство только для чтения.

 

Синтаксис

object.TreeId

Синтаксис свойства TreeId состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево

 

Тип данных

Строка

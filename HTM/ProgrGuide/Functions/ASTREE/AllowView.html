Свойство AllowView

См. также  Пример  Применяется к

Возвращает разрешение на просмотр документа в дереве.

Свойство только для чтения.

 

Синтаксис

object.AllowView

Синтаксис свойства AllowView состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево

 

Тип данных

Логический

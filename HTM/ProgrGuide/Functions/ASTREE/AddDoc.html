Метод AddDoc

См. также  Пример  Применяется к

Добавляет возможность создания нового типа документа в дереве.

 

Синтаксис

object.AddDoc sDocID

Синтаксис метода AddDoc состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерева.
sDocID строковое выражение, определяющее идентификатор документа

 

Примечание

После применения данного метода, в дереве можно создать новый экземпляр документа указанного типа из меню или из контекстного меню.

Метод DeleteNode

См. также  Пример  Применяется к

Удаляет ветвь из дерева.

 

Синтаксис

object.DeleteNode sCode

Синтаксис метода DeleteNode состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерева.
sCode строковое выражение, определяющее код удаляемой ветви.

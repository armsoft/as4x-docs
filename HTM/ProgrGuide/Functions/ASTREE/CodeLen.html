Свойство CodeLen

См. также  Пример  Применяется к

Возвращает или устанавливает длину кода дерева.

Свойство для чтения и записи.

 

Синтаксис

object.CodeLen

Синтаксис свойства CodeLen состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево


Тип данных

Целое число

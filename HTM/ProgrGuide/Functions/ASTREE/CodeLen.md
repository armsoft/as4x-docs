# CodeLen հատկություն

[См. также](../Astree.html) [Օրինակ](../../Examples/E_AsTree.html) [Применяется к](../Astree.md)

Վերադարձնում կամ նշանակում է ծառի կոդի երկարությունը։


## Շարահյուսություն

``` vb
object.CodeLen [= newValue]
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object | Ծառի օբյեկտի հղում։|

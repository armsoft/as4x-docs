Метод ClearAllDocs

См. также  Пример  Применяется к

Удаляет возможность добавления документа дереве.

 

Синтаксис

object.ClearAllDocs

Синтаксис метода ClearAllDocs состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево.

 

Примечание

Данный метод удаляет описания всех документов из дерева. Это значит, что после его применения в дереве будет невозможно добавление какого-либо типа документа. На панели управления иконка "New" станет недоступным.

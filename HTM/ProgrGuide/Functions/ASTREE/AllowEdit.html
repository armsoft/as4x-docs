Свойство AllowEdit

См. также  Пример  Применяется к

Возвращает разрешение на корректировку документа в дереве.

Свойство только для чтения.

 

Синтаксис

object.AllowEdit

Синтаксис свойства AllowEdit состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево

 

Тип данных

Логический

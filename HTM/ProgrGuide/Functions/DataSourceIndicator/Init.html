Метод Init

См. также  Пример  Применяется к

Инициализирует скриптовую процедуру из заданного модуля.

Свойство для чтения и записи.

 

Синтаксис

object.Init [modName, subName] 

Синтаксис свойства Init состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа DataSourceIndicator.
modName строковое выражение, определяющее имя модуля.
subName строковое выражение, определяющее имя процедуры.

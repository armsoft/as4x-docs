Свойство Properties

См. также  Пример  Применяется к

Возвращает ссылку на объект типа Dictionary. Данное свойство применяется для хранения некоторых значений для последующей их обработки.

Синтаксис

object.Properties

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа DataSourceIndicator.

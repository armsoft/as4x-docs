Метод Run

См. также  Пример  Применяется к

Выполняет скриптовую процедуру для заданного источника данных. 

 

Синтаксис

object.Run[xData]

Синтаксис свойства Run состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа DataSourceIndicator.
xData объект типа источника данных.

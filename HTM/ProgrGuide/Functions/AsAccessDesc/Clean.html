Метод Clean

См. также  Пример  Применяется к

Создает пустой дескриптор доступа для указанного объекта.

 

Синтаксис

object.Clean (nISN, btAccessID)

Синтаксис метода Clean состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект описатель доступа.
nISN длинное целое, определяющее ISN объекта, для которого создается дескриптор доступа.
btAccessID байтовое выражение, определяющее код доступа.

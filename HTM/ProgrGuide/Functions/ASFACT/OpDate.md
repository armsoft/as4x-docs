# OpDate հատկություն

[Տես նաև](../Asfact.md) [Օրինակ](../../Examples/E_AsFact.md) [Կիրառվում է](../Asfact.md)

Վերադարձնում կամ նշանակում է գործողության ամսաթիվը։

## Շարահյուսություն

```vb
object.OpDate [=dValue] 
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Գործառնություն օբյեկտի հղում։ |
| dValue | Նոր արժեք։ |

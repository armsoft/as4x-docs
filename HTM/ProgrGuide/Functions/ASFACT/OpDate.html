Свойство OpDate

См. также  Пример  Применяется к

Возвращает или устанавливает дату операции.

Свойство для чтения и записи.

 

Синтаксис

object.OpDate [=dValue]   

Синтаксис свойства OpDate состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
dValue выражение типа даты, определяющее дату операции проводки

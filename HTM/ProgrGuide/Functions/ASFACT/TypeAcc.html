Свойство TypeAcc

См. также  Пример  Применяется к

Возвращает код учета для данной проводки.

Свойство только для чтения.

 

Синтаксис

object.TypeAcc

Синтаксис свойства TypeAcc состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
sValue строковое выражение, определяющее код учета для проводки. Максимальная длина кода учета 2 символа.

 

Тип данных

Строка

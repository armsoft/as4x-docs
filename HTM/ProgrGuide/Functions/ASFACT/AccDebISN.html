Свойство AccDebISN

См. также  Пример  Применяется к

Возвращает или устанавливает внутрисистемный код ISN для дебетуемого объекта.

Свойство для чтения и записи.

 

Синтаксис

object.AccDebISN = [nISN]

Синтаксис свойства AccDebISN состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
nISN переменная типа Long, определяющая значение внутрисистемного кода ISN для дебетуемого объекта.

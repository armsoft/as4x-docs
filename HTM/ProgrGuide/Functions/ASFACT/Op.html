Свойство Op

См. также  Пример  Применяется к

Возвращает или устанавливает код операции.

Свойство для чтения и записи.

 

Синтаксис

object.Op [=sValue]   

Синтаксис свойства Op состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
sValue строковое выражение, определяющее код типа операции в проводке. Максимальная длина строки 3 символа.

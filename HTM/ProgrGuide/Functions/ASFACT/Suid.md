# Suid հատկություն

[Տես նաև](../Asfact.md) [Օրինակ](../../Examples/E_AsFact_SUID.md) [Կիրառվում է](../Asfact.md)

Վերադարձնում կամ նշանակում է գործառնության օգտագործողի ներքին համարը։

## Շարահյուսություն

```vb
object.Suid  = [nSuid]
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Գործառնություն օբյեկտի հղում։ |
| nSuid | Նոր արժեք։ |

## Տվյալի տիպ

Ամբողջ թիվ

Свойство AccCrd

См. также  Пример  Применяется к

Возвращает или устанавливает кредитуемый объект проводки.

Свойство для чтения и записи.

 

Синтаксис

object.AccCrd [=vValue]   

Синтаксис свойства AccCrd состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
vValue выражение типа Variant, определяющее кредитуемый объект, в частности счет.

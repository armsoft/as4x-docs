Свойство Summa

См. также  Пример  Применяется к

Возвращает или устанавливает для проводки сумму в национальной денежной единице. После задания суммы, ядро системы автоматически округляет его до заданной степени округления из описания учета.

Свойство для чтения и записи.

 

Синтаксис

object.Summa = [nSum]

Синтаксис свойства Summa состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
nSum переменная типа Currency, определяющая сумму проводки в национальной денежной единице.

# Cur հատկություն

[Տես նաև](../Asfact.md) [Օրինակ](../../Examples/E_AsFact.md) [Կիրառվում է](../Asfact.md)

Վերադարձնում կամ նշանակում է գործառնության արտարժույթի կոդը։

## Շարահյուսություն

```vb
object.Cur [= nValue]  
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրություն |
|--|--|
| object | Գործառնություն օբյեկտի հղում։ |
| nValue | Նոր արժեք։ |

Свойство ObjectISN

См. также  Пример  Применяется к

Возвращает или устанавливает внутрисистемный код ISN аккумулирующего объекта.

Свойство для чтения и записи.

 

Синтаксис

object.ObjectISN = [nISN]

Синтаксис свойства ObjectISN состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
nISN переменная типа Long, определяющая значение внутрисистемного кода ISN для объекта проводки.

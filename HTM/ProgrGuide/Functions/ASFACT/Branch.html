Свойство Branch

См. также  Пример  Применяется к

Возвращает или устанавливает значение кода филиала проводки. Максимально допустимая длина 3 символов.

Свойство для чтения и записи.

 

Синтаксис

object.Branch [=sValue]   

Синтаксис свойства Branch состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
sValue строковое выражение.

Свойство Accounting

См. также  Пример  Применяется к

Возвращает ссылку на объект типа учет для текущей проводки.

Свойство для чтения.

 

Синтаксис

object.Accounting

Синтаксис свойства Accounting состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка

Свойство Suid

См. также  Пример  Применяется к

Возвращает или устанавливает код оператора для данной проводки.

Свойство для чтения и записи.

 

Синтаксис

object.Suid  = [nSuid]

Синтаксис свойства Suid состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
nSuid строковое выражение, определяющее код оператора.

 

Тип данных

Целое число

Свойство Visible

См. также  Пример  Применяется к

Возвращает или устанавливает признак показа проводок при сохранении документа регистрирующего факты.

Свойство для чтения и записи.

 

Синтаксис

object.Visible [=bValue]   

Синтаксис свойства Visible состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
bValue логическое выражение, определяющее признак показа проводки при регистрации фактов

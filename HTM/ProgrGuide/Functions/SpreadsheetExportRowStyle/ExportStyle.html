 Функция ExportStyle

свойства   методы

Данная функция возвращает объект типа SpreadsheetExportRowStyle по заданному имени стиля.

Синтаксис

object.ExportStyle(StyleName)

Синтаксис метода ExportStyle состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
StyleName строковое выражение, определяющее имя стиля экспорта строк справки в Excel. Если данный параметр отсутствует или по заданному имени не было определено ни одного стиля, то возвращаемое значение будет равно Nothing,  в противном случае функция возвращает объект уже существующего стиля по заданному имени.

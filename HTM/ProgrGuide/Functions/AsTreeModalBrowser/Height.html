Свойство Height

 См. также  Пример  Применяется к

Возвращает или устанавливает высоту окна иерархического вспомогательного списка.

Свойство для чтения и записи.

 

Синтаксис

object.Height = [nHeightValue]

Синтаксис свойства Height состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.
nHeightValue численное выражение типа Single, определяющее высоту появляющегося окна.

 

Тип данных

Single

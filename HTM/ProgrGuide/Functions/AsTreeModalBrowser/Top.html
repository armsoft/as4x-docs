Свойство Top

 См. также  Пример  Применяется к

Возвращает или устанавливает удаленность верхнего края иерархического вспомогательного списка от верхнего края окна приложения.

Свойство для чтения и записи.

 

Синтаксис

object.Top = [nTopValue]

Синтаксис свойства Top состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.
nTopValue выражение типа Single, определяющее расстояние между верхним краем иерархического вспомогательного списка и верхним краем окна приложения.

 

Тип данных

Single

Свойство isSelected

 См. также  Пример  Применяется к

Возвращает признак того, выбрана ли строка из иерархического вспомогательного списка.

Свойство для чтения.

 

Синтаксис

object.isSelected

Синтаксис свойства isSelected состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.

 

Тип данных

Логический

Свойство Value

 См. также  Пример  Применяется к

Возвращает ключ в дереве выбранного элемента иерархического вспомогательного списка. Если, конечно, осуществлен выбор и значение свойства isSelected=True.

Свойство для чтения.

 

Синтаксис

object.Value

Синтаксис свойства Value состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.

 

Тип данных

Строка

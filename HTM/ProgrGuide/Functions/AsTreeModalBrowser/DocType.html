Свойство DocType

 См. также  Пример  Применяется к

Возвращает тип документа выбранного элемента иерархического вспомогательного списка. Если, конечно, осуществлен выбор и значение свойства isSelected=True.

Свойство для чтения.

 

Синтаксис

object.DocType

Синтаксис свойства DocType состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.

 

Тип данных

Строка

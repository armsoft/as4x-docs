Свойство Left

 См. также  Пример  Применяется к

Возвращает или устанавливает удаленность левого края иерархического вспомогательного списка от левого края окна  приложения.

Свойство для чтения и записи.

 

Синтаксис

object.Left = [nLeftValue]

Синтаксис свойства Left состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.
nLeftValue численное выражение типа Single, определяющее расстояние между левым краем иерархического вспомогательного списка и левым краем окна приложения.

 

Тип данных

Single

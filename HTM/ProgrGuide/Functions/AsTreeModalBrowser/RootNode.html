Свойство RootNode

 См. также  Пример  Применяется к

Возвращает или устанавливает корневой элемент объекта типа иерархический вспомогательный список. Данное свойство служит для показа поддерьевьев.

Свойство для чтения и записи.

 

Синтаксис

object.RootNode= [sValue]

Синтаксис свойства RootNode состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа иерархический вспомогательный список.
sValue строковое выражение, определяющее ключ элемента в дереве

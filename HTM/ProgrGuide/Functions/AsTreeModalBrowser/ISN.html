Свойство ISN

 См. также  Пример  Применяется к

Возвращает внутрисистемный уникальный идентификатор документа выбранного элемента иерархического вспомогательного списка. Если, конечно, осуществлен выбор и значение свойства isSelected=True.

Свойство для чтения.

 

Синтаксис

object.ISN

Синтаксис свойства ISN состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.

 

Тип данных

Длинное целое

Свойство GotBreakSignal

 См. также  Пример  Применяется к

Возвращает признак нажатия клавиши прерывания расчетного процесса.

 

Синтаксис

object.GotBreakSignal

Синтаксис свойства GotBreakSignal состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа обновитель экрана

 

Тип данных

Логический

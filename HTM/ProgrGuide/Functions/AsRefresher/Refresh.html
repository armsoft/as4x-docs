Метод Refresh

 См. также  Пример  Применяется к

Перерисовывает окно приложения. Полезен в случаях переключения между окнами активных приложений Windows во время расчетных процессов. Метод применяется в контрольных точках циклов расчета по необходимости.

 

Синтаксис

object.Refresh

Синтаксис метода Refresh состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа обновитель экрана

# WeekDay հատկություն
[См. также](../AsDayStatus.md) Օրինակ [Применяется к](../AsDayStatus.md)

Վերադարձնում է շաբաթվա օրը։ 

Կարդալու հատկություն։


## Շարահյուսություն

``` vb
object.WeekDay
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| object | Օրվա կարգավիճակի հղում։ |

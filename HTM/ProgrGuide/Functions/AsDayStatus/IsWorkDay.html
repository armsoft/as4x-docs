Свойство IsWorkDay

См. также  Пример  Применяется к

Возвращает признак того является ли день рабочым или нет.

Свойство для чтения.

 

Синтаксис

object.IsWorkDay

Синтаксис свойства IsWorkDay состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр статуса дня

 

Тип данных

Логический

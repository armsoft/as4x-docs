Объект коллекция файловых дополнений

Каждый документ может содержать коллекцию файловых дополнений, если в определении данного документа задан признак приклепления. Элементами данной коллекции являются объекты типа файловое дополнение.

Свойства Описание
Count Возвращает количество элементов в коллекции файловых дополнений.
Item

Возвращает элемент из коллекции файловых дополнений по ключу.

 

Методы Описание
Add Добавляет новый элемент в коллекцию файловых дополнений.
Refresh Обновляет коллекцию файловых дополнений, загружая их из сервера.
Remove Удаляет файловое дополнение из коллекции.

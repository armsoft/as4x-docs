Метод GetBrowseParametersReport

См. также  Пример  Применяется к

Создает объект типа отчет, в котором содержится информация о параметрах  диалога-фильтра вида просмора.

 

Синтаксис

object.GetBrowseParametersReport()

Синтаксис свойства GetBrowseParametersReport состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта  AsBrowserParametersInfo

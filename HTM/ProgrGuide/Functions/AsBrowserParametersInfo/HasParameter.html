Свойство HasParameter

См. также  Пример  Применяется к

Возвращает признак существования заданного параметра в диалоге. Если параметр существует, то возвращаемое значение будет True, в противним случае False.

 

Синтаксис

object.HasParameter(sName)

Синтаксис свойства HasParameter состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта   AsBrowserParametersInfo
sName строковое выражение, определяющее идентификатор проверяемого на существование параметра

Метод AddDSParameter

См. также  Пример  Применяется к

Создает параметр для источника данных вида просмора.

 

Синтаксис

object.AddDSParameter(sName, sCaption, sType, sValue)

Синтаксис свойства AddDialogParameter состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта AsBrowserParametersInfo
sName строковое выражение, определяющее идентификатор  добавляемого параметра
sCaption строковое выражение, определяющее наименование  добавляемого параметра
sType строковое выражение, определяющее тип  добавляемого параметра
sValue строковое выражение, определяющее текстовое значение параметра

Свойство DialogValue

См. также  Пример  Применяется к

Возвращает значение параметра из диалога.

 

Синтаксис

object.DialogValue(sName)

Синтаксис свойства DialogValue состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объектаAsBrowserParametersInfo
sName строковое выражение, определяющее идентификатор реквизита диалога-фильтра вида просмотра

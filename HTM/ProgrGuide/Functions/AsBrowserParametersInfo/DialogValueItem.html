Свойство DialogValueItem

См. также  Пример  Применяется к

Возвращает признак наличия  параметра диалога-фильтра из списка выбранных значений метода пользовательского диалога  AddCheckedListBox. Если проверяемое значение присутствует в списке, то DialogValueItem возвращает значение True, в противном случае False. 

Синтаксис

object.DialogValueItem(sName, sItemKey)

Синтаксис свойства DialogValueItem состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта  AsBrowserParametersInfo
sName строковое выражение, определяющее идентификатор параметра диалога-фильтра вида просмотра
sItemKey строковое выражение, определяющее проверяемое значение реквизита диалога-фильтра вида просмотра

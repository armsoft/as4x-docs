Объект учет

Учет должен быть заранее описан и загружен в систему. Доступ к описанию учета обеспечивается через функцию Accounting.

Свойства Описание
AccFolder Возвращает идентификатор папки счетов из описания учетов.
Caption Возвращает заголовок учета.
CountOperation Возвращает количество операций описанных в учете.
Cur Возвращает признак использования валюты в заданном учете.
Description Возвращает краткий разъяснительный текст учета из его описания.
FactHeader Возвращает или устанавливает трафарет проводок в окне предварительного показа проводок и остатков в заданном учете.
LastFactDate Возвращает дату последней проводки в заданном учете.
Limit Возвращает признак автоматической проверки лимитов по объектам учета.
MainFolder Возвращает идентификатор папки объектов над которыми в указанном учете производятся операции.
Name Возвращает идентификатор учета из его описания.
Remainder Возвращает разрешение на ведение учета с отслеживанием остатков.
RemHeader Возвращает или устанавливает трафарет остатков в окне предварительного показа проводок и остатков в заданном учете.
RoundSumma Возвращает степень округления суммы в НДЕ для проводок данного учета.
RoundCurSumma Возвращает степень округления валютной суммы для проводок данного учета.

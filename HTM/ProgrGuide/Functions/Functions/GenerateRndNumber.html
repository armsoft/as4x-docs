Функция GenerateRndNumber

Возвращает случайно сгенерируемые числа.

Возвращаемый тип  значения целое число.

 

Синтаксис

GenerateRndNumber(nDecimalLength, nMinNumber )

Синтаксис функции GenerateRndNumber состоит из следующих частей:

Параметр Описание
nDecimalLength обязательное численное выражение, определяющее конечный предел инервала, внутри которого генерируется число.
nMinNumber обязательное численное выражение, определяющее начальное значение интервала, в пределах которого генерируется число.

 

Примечание

   Предельные значения допустимого интервала сгенерируемого числа  = [nMinNumber, 10^nDecimalLength].

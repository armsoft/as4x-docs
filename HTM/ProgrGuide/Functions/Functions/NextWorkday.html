Функция NextWorkDay

Возвращает ближайший следующий рабочий день.

Возвращает значение типа дата.

 

Синтаксис

NextWorkday (dDay)                                                                                                     

Синтаксис функции NextWorkday состоит из следующих частей:

Параметр Описание
dDay выражение типа дата, определяющее дату после которой возвращается ближайший следующий рабочий день.

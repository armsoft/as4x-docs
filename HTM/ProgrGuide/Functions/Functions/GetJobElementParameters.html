Функция GetJobElementParameters

Возвращает параметры задания.

Возвращаемое значение объект типа XArrayD.

 

Синтаксис

GetJobElementParameters(sJobElementName)

Синтаксис функции GetJobElementParameters состоит из следующих частей:

Параметр Описание
sJobElementName строковое выражение, определяющее идентификатор задания

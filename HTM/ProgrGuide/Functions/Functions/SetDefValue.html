Метод SetDefValue

Задает значения по умoлчанию заданным реквизитам.

Синтаксис

SetDefValue(docOrDlg, rekvsList)

Синтаксис функции SetDefValue состоит из следующих частей:


Параметр Описание
docOrDlg Обязательная переменная, ссылающаяся на экземпляр объекта типа документ или на экземпляр объекта типа описательный диалог.
rekvsList Обязательное строковое выражение, определяющее список реквизитов, разделенных пробелами.

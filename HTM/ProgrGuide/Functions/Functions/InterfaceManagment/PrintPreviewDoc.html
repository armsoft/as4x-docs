Метод PrintPreviewDoc

Показывает печатную форму документа(ов).

 

Синтаксис

PrintPreviewDoc (xDoc, [iPrintMode])

Синтаксис метода PrintPreviewDoc состоит из следующих частей:

Параметр Описание
xDoc выражение типа Variant, возможны следущие значения этого параметра:
1. Экземпляр документа .
2. ISN докмента.
3. Массив документов.
4. Массив ISN документов.
iPrintMode необязательное выражение типа Variant, определяющее режим печати. При отсутствии данного параметра принимается режим печати по умолчанию.

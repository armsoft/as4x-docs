Свойство MousePointer

Устанавливает значение для внешнего вида указателя мыши. Применяется для индикации длительных расчетных процессов.

Свойство для записи.

 

Синтаксис

MousePointer=nValue

Установки для nValue следующие:

Константа Значение Описание
vbHourGlass 11 указатель мыши принимает вид песочных часов
vbDefault 0 указатель мыши принимает обычный вид стрелки

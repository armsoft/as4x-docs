Функция DocP
свойства   методы

Возвращает ссылку на объект печатного вида документа. Данная функция может использоваться только в скриптовых процедурах Print, PrintExcel, PrintWord в описании документа.

 

Синтаксис

set sDocP = DocP

Синтаксис функции DocP состоит из следующих частей:

Параметр Описание
sDocP строковое выражение, определяющее переменную, ссылающуюся на объект печатной формы. Через нее можно получить доступ к свойствам и методам объекта.

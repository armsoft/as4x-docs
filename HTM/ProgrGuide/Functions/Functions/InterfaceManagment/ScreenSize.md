# ScreenSize Ֆունկցիա

Եթե չափը 800/600 -ից մեծ է, ապա վերադարձնում է 1, հակառակ դեպքում՝ 0:

Возвращает разрешение экрана. Если разрешение больше 800/600, то возвращает 1, в противном случае - 0.


## Շարահյուսություն

```vb
ScreenSize
```

Функция AccountingArchiveDate

Возвращает дату начиная с которой данные пернесены в архив для данного учета.

Возвращает значение типа Date.

 

Синтаксис

AccountingArchiveDate (Acc)

Синтаксис функции AccountingArchiveDate состоит из следующих частей:

Параметр Описание
Acc строковое выражение, определяющее код учета.

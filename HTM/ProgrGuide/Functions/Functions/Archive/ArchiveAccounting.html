Метод ArchiveAccounting

Делает архивацию таблицы HI для данного учета.

 

Синтаксис

Util.ArchiveAccounting(Acc, ArchiveDate)

Синтаксис метода ArchiveAccounting состоит из следующих частей:

Параметр Описание
Acc строковое выражение, определяющее код учета.
ArchiveDate выражение типа Date, определяющее дату до которого включительно происходит архивация.

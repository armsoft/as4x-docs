Метод ArchiveAccountingHI2

Делает архивацию таблицы HI2 для данного учета.

 

Синтаксис

Util.ArchiveAccountingHI2(Acc, ArchiveDate)

Синтаксис метода ArchiveAccountingHI2 состоит из следующих частей:

Параметр Описание
Acc строковое выражение, определяющее код учета.
ArchiveDate выражение типа Date, определяющее дату до которого включительно происходит архивация.

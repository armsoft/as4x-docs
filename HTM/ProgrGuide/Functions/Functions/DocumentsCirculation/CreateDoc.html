Функция CreateDoc

свойства  методы

Создает новый экземпляр документа указанного типа.

Синтаксис

Set sNewDoc = CreateDoc(sDocType, [nParentISN])

Синтаксис функции CreateDoc состоит из следующих частей:

Параметр Описание
sNewDoc переменная, ссылающаяся на новый экземпляр документа
sDocType строковое выражение, определяющее тип создаваемого документа
nParentISN необязательное численное выражение типа Long, определяющее ISN родительского документа. По умолчанию равно -1.

Пример функции CreateDoc

В примере вызывается функция CreateDoc, xDoc ссылается на обьект типа документ со всеми его свойствами и методами.
Создается новый экземпляр документа типа HTN92. 

Set mDoc = LoadDoc(CurrentView.Value("fISN"))
Set xDoc = CreateDoc("HTN92", mDoc.ISN)
xDoc("CATEGORY") = xMsgType
xDoc("BMDOCNUM") = nDoc.ISN
xDoc("REFERENC") = mDoc("BMDOCNUM")
xDoc("DATE") = mDoc("BMIODATE")
xDoc("TYPE") = xMsgType
xDoc.State = 0
xDoc.Show

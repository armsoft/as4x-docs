Метод CutChildLink

Для указанного документа разрывает связь с потомком.

Синтаксис

CutChildLink (nISN, [nChildISN])

Синтаксис метода CutChildLink состоит из следующих частей:

Параметр Описание
nISN численное выражение типа Long, определяющее ISN документа, для которого разрывается связь с потомком.
nChildISN необязательное численное выражение типа Long, определяющее ISN документа потомка, с которым разрывается связь. Если параметр опущен, то разрываются связи со всеми существующими потомками.

Примечание

Из таблицы DOCP удаляется строка, соответствующая данному документу.

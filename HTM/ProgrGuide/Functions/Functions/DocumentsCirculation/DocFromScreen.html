Функция DocFromScreen

свойства   методы

Возвращает ссылку на еще несохраненный  экземпляр документа с экрана.

Синтаксис

Set sDoc = DocFromScreen (nISN)

Синтаксис функции DocFromScreen состоит из следующих частей:

Параметр Описание
sDoc переменная, ссылающаяся на экранный несохраненный документ
nISN численное выражение типа Long, определяющее ISN загруженного документа. Если задается несуществующий ISN, то выдается Nothing.

Примечание

Функция DocFromScreen выполняет то же самое действие, что и LoadDoc. Отличие их состоит в том, что LoadDoc возвращает ссылку на существующий в базе документ, а функция DocFromScreen возвращает ссылку на несохраненный, находящийся еще на экране документ.
Данная функция полезна в следующем случае. Например создается документ, который еще не сохранен и при его сохранении рождаются несколько других документов-потомков. Документы потомки нуждаются в значениях еще несохраненного родительского документа. Ссылка на родительский документ создается с помощью функции DocFromScreen, а не LoadDoc.

Пример функции DocFromScreen

В примере вызывается функция DocFromScreen, xParent ссылается на несохраненный еще экранный документ со всеми его свойствами и методами. Из скрипта для документа потомка используются значения реквизитов родительского документа. 

Sub Folders()
  Set xParent = DocFromScreen(Doc.ParentISN)
  Doc.PutFolder "CREDCHILDS", "O", Doc("CODE"), xParent("NAME"), _
    xParent("CODVAL") & xParent("CLICOD")
End Sub

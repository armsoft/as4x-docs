Функция PageCaption

Возвращает заголовок страницы документа, зафиксированный в классе ASDocDef при описаниии документа.

Возвращаемое значение строка.

 

Синтаксис

PageCaption(PageName)

Синтаксис функции PageCaption состоит из следующих частей:

Параметр Описание
PageName строковое выражение, определяющее наименование страницы документа, заголовок которого будет возвращено.

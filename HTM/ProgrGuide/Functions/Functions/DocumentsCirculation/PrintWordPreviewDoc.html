Метод PrintWordPreviewDoc

Создает форму документа в программе MS Word.

 

Синтаксис

 PrintWordPreviewDoc DocOrISN

Синтаксис функции PrintWordPreviewDoc состоит из следующих частей:

Параметр Описание
DocOrISN выражение типа варианты, определяющее: либо внутрисистемный идентификационный код ISN документа, либо же экземпляр документа..

Функция CreateFolderElement

свойства  методы

Создает новый экземпляр элемента папки для дальнейшего заполнения и запоминания.

Синтаксис

Set sNewFolderEl = CreateFolderElement(FolderId)

Синтаксис функции CreateFolderElement состоит из следующих частей:

Параметр Описание
sNewFolderEl переменная, ссылающаяся на новый экземпляр элемента папки
FolderId строковое выражение, определяющее идентификатор папки, для которой создается элемент

Пример функции CreateFolderElement

В примере вызывается функция CreateFolderElement, xFoldEl ссылается на вновь созданный объект-проводку со всеми его свойствами и методами. 

Set xFoldEl = CreateFolderElement("NBACC")
xFoldEl.Com = Doc("Name")
xFoldEl.Ecom = Doc("EName")
xFoldEl.Spec = Doc("NOMDOK") & Doc.Formatted("MTCCOUNT") & Doc("COMENT") 
xFoldEl.Status = "E"
Doc.StoreInFolder xFoldEl

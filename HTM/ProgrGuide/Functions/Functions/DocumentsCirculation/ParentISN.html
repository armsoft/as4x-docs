Функция ParentISN

Возвращает ISN родительского документа для указанного потомка. Если родитель отсутствует то возвращается -1.

Возвращает численное значение типа Long.

 

Синтаксис

ParentISN (nISN)

Синтаксис функции ParentISN состоит из следующих частей:

Параметр Описание
nISN численное выражение, определяющее ISN документа-потомка для искомого документа-родителя.

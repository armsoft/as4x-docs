Метод CreateParentLink

Создает родительскую связь между документами.

 

Синтаксис

CreateParentLink (ISN, ParentISN)

Синтаксис метода CreateParentLink состоит из следующих частей:

Параметр Описание
ISN обязательное численное выражение типа Long, определяющее ISN документа, для которого создается родительская связь.
ParentISN обязательное численное выражение типа Long, определяющее ISN родительского документа

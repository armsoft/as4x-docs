Функция CopyDoc

свойства  методы

Создает копию существующего экземпляра документа.

Синтаксис

Set sDoc = CopyDoc (nISN, [ByVal BeforeCopyParam As Variant], [ByVal Mode As Integer])

Синтаксис функции CopyDoc состоит из следующих частей:

Параметр Описание
sDoc переменная, ссылающаяся на создаваемый документ
nISN численное выражение типа Long, определяющее ISN копируемого документа.
BeforeCopyParam необязательное выражение типа Variant, значение этого параметра передается событию документа BeforeCopy.
Mode необязательное выражение типа Integer, определяющее режим копирования документов:
0 - Значение всех реквизитов копируется.
1 - Режим копирования соответствует визуальному копированию (зависит от значения CopyAsRepeatable в описании документа.
2 - Режим, при котором копируются только реквизиты, содержащие атрибут N.
По умолчанию принимает значение 0.

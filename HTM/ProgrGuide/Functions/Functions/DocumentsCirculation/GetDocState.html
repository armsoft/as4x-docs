Функция GetDocState

Возвращает состояние указанного документа. Если документ не существует, либо документ не окончательно удален, а только помечен как удаленный (DeleteDoc), то функция возвращает значение (-1).

Возвращает численное значение типа Long.

 

Синтаксис

GetDocState( ISN )

Синтаксис функции GetDocState состоит из следующих частей:

Параметр Описание
ISN численное выражение типа Long, определяющее внутрисистемный идентификационный код (ISN) документа, для которого определяется состояние.

Метод UserPositionAndMask

Возвращает номер позиции и маску пользователя системы.

Синтаксис

UserPositionAndMask(ByRef  nNumber, ByRef  nMask,  [mySUID])

Синтаксис функции UserPositionAndMask состоит из следующих частей:

Параметр Описание
nNumber

Переменная типа байт, определяющая позицию пользователя системы.

nMask Переменная типа байт, определяющая маску пользователя системы.
mySUID Необязательная  переменная целого типа, определяющая код пользователя.

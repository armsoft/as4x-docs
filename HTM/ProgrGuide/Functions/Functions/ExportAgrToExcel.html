Метод ExportAgrToExcel

Данный метод экспортирует заданный вид просмотра в Excel.

 

Синтаксис

ExportAgrToExcel (viewObject)

Синтаксис метода ExportAgrToExcel состоит из следующих частей:

Параметр Описание
viewObject обязательное выражение, являющееся ссылкой на экземпляр объекта вида просмотра

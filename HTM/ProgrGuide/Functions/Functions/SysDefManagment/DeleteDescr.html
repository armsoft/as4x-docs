Метод DeleteDescr

Удаляет системное описание из таблицы SYSDEF.

 

Синтаксис

Util.DeleteDescr (nSysType, sDefID)

Синтаксис метода DeleteDescr состоит из следующих частей:

Параметр Описание
nSysType численное выражение целого типа, определяющее внутрисистемный код типа описания. Его значения аналогичны допустимым значениям поля fSYSTYPE в таблице SYSDEF, кроме значения 5 (описание навигаторского дерева), которое   удаляется интерактивно.
sDefID строковое выражение, определяющее идентификатор удаляемого описания.

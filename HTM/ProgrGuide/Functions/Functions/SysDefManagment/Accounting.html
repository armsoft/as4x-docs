Функция Accounting
свойства   методы

Возвращает ссылку на объект типа учет.

 

Синтаксис

set sUchObj = Accounting(CodUch)

Синтаксис функции Accounting состоит из следующих частей:

Параметр Описание
sUchObj переменная, ссылающаяся на объекта типа учет. Через нее можно получить доступ к свойствам и методам объекта.
CodUch строковое выражение, определяющее код загружаемого учета

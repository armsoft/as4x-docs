Функция SpaceSplit

Возвращает одномерный массив заданной длины, разбитый на подстроки.

Тип возвращаемого значения вариант

 

Синтаксис

SpaceSplit(strArr2Split, length1, lengthRem)

Синтаксис функции SpaceSplit состоит из следующих частей:

Параметр Описание
strArr2Split выражение типа вариант, определяющее исходный массив, который необходимо разбить на подстроки.
length1 численное выражение, определяющее максимальную  длину первого элемента  массива.
lengthRem численное выражение, определяющее максимальную  длину остальной части  массива без первичного элемента.

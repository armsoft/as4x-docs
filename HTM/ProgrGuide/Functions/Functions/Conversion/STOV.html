Функция STOV

Преобразовывает заданную строку в значение внутреннего представления.

Возвращается значение типа Variant.

 

Синтаксис

STOV (strType, sValue)

Синтаксис функции STOV состоит из следующих частей:

Параметр Описание
strType строковое выражение, определяющее внутренний тип данных для преобразования.
Value строковое выражение, определяющее исходное значение, подлежащее преобразованию.

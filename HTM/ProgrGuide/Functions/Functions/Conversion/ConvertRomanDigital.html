Функция ConvertRomanDigital

Преобразовывает заданное число в римские цифры.

 

Тип возвращаемого значения строка.

 

Синтаксис

ConvertRomanDigital (convNumber)

Синтаксис функции ConvertRomanDigital состоит из следующих частей:

Параметр Описание
convNumber обязательное численное выражение, которую нужно конвертировать.

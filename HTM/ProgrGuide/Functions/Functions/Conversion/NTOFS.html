Функция NTOFS

Преобразовывает заданное число в форматированное строковое представление определенной длины с позиционными разделительными запятыми.

Возвращается символьная строка.

 

Синтаксис

NTOFS (strType, Value)

Синтаксис функции NTOFS состоит из следующих частей:

Параметр Описание
strType строковое выражение, определяющее целевой численный тип (N(), NP(), Summa, SummaP) для преобразования. Исходя из длины типа определяется длина форматированной строки, учитывая при этом количество позиционных разделительных запятых.
Value выражение типа Variant, определяющее исходное значение числа, подлежащее преобразованию в заданный тип.

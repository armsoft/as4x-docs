Функция GetNumber2

Возвращает второе число из значения типа числовая пара.

Возвращаемое значение типа Currency.

 

Синтаксис

GetNumber2(sNumPair)

Синтаксис функции GetNumber2 состоит из следующих частей:

Параметр Описание
sNumPair строковое выражение, определяющее строку типа "число1 / число2" из которой функция возвращает число2.

Функция DefaultValue

Возвращает для указанного типа значение по умолчанию.

 

Синтаксис

DefaultValue (sTypeName)

Синтаксис функции DefaultValue состоит из следующих частей:

Параметр Описание
sTypeName строковое выражение, определяющее внутренний тип данных


Ниже приведена таблица типов со значениями по умолчанию:

Тип данных Значение по умолчанию
AmAcc Param("CODBANK")+Space(11)
Boolean 0
C () Строка, состоящая из пробелов в количестве, указанном в типе
Ch () Строка, состоящая из пробелов в количестве длины ключа
Date Null
DateLong Null
DateRep Param("REPSTART")
Folder () Строка, состоящая из пробелов длиной равной длине ключа указанной папки.
FullTree () если указанное дерево существует, то возвращается строка пробелов c длиной равной длине кода дерева, а если дерево не существует, то возвращается Null.
N () 0
NP () 0
NumPair() "0 / 0"
Summa 0
SummaP 0
Tree () если указанное дерево существует, то возвращается строка пробелов c длиной равной длине кода дерева, а если дерево не существует, то возвращается Null.

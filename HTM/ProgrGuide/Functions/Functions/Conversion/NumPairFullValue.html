Функция NumPairFullValue

Для заданных двух чисел формирует значение типа числовая пара. Функция полезна для формирования значений, подставляемых в реквизиты типа NumPair().

Возвращаемое значение строка.

 

Синтаксис

NumPairFullValue(nNum1, nNum2)

Синтаксис функции NumPairFullValue состоит из следующих частей:

Параметр Описание
nNum1 численное выражение, определяющее первое число для образования строки типа "число1 / число2".
nNum2 численное выражение, определяющее второе число для образования строки типа "число1 / число2".

 

Примечание

Обратное выделение чисел из составной строки производится с помощью функций GetNumber1, GetNumber2.

Функция ConvertArmenianANSIToUnicode

Преобразовывает заданную строку на aрмянском языке из ANSI в UNICODE .

Тип возвращаемого значения строка.

 

Синтаксис

ConvertArmenianANSIToUnicode(armSource)

Синтаксис функции ConvertArmenianANSIToUnicode состоит из следующих частей:

Параметр Описание
armSource строковое выражение, которую нужно конвертировать.

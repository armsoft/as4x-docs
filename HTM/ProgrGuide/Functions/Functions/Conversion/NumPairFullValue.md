# NumPairFullValue ֆունկցիա

Տրված երկու թվերի համար ձևավորվում է [թվային զույգ](../../../Types/NumPair.md) տիպի արժեքը։ Այս Ֆունկցիան հարմար է արժեքների ձևավորման համար, որոնք փոխարկվում են [NumPair()](../../../Types/NumPair.html) տիպի դաշտի հետ։ 

Վերադարձվող արժեքը 
Для заданных двух чисел формирует значение типа [числовая пара](../../../Types/NumPair.html). Функция полезна для формирования значений, подставляемых в реквизиты типа [NumPair()](../../../Types/NumPair.html).<br>
<br>
Возвращаемое значение строка.



## Շարահյուսություն

```vb
NTOFS ֆունկցիա
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| nNum1 | Սահմանում է առաջին թիվը ["թիվ1 / թիվ2"](../../../Types/NumPair.md) տողը ձևավորելու համար։ численное выражение, определяющее первое число для образования строки [типа &quot;число1 / число2&quot;](../../../Types/NumPair.html). |
| nNum2 | Սահմանում է երկրորդ թիվը ["թիվ1 / թիվ2"](../../../Types/NumPair.md) տողը ձևավորելու համար։ численное выражение, определяющее второе число для образования строки [типа &quot;число1 / число2&quot;](../../../Types/NumPair.html). |


## Նկատառումներ

Թվերի հակառակ առանձնացումը բարդ տողից կատարվում է [GetNumber1](GetNumber1.md) և [GetNumber2](GetNumber2.md) ֆունկցիաների միջոցով։
Обратное выделение чисел из составной строки производится с помощью функций [GetNumber1](GetNumber1.html), [GetNumber2](GetNumber2.html).<br>

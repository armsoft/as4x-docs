Функция LeftAlign

Выравнивает строку по левому краю добавляя недостающие пробелы (или другие указанные символы) справа. Функция удобна для получения строк фиксированной длины для использования в отчетах. К исходной строке (функция предварительно убирает лишние пробелы слева от исходного значения) справа добавляются недостающие символы для получения указанной в аргументе длины.

Возвращает строку.

 

Синтаксис

LeftAlign(sSource, intLength, [sDelimiter])

Синтаксис функции LeftAlign состоит из следующих частей:

Параметр Описание
sSource строковое выражение, определяющее исходную строку
intLength численное выражение целого типа, определяющее длину выравненной результирующей строки.
sDelimiter необязательное строковое выражение, определяющее тип заполняемых недостающих символов справа от строки. По умолчанию это пробелы.

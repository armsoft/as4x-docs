Функция GetNumber1

Возвращает первое число из значения типа числовая пара.

Возвращаемое значение типа Currency.

 

Синтаксис

GetNumber1(sNumPair)

Синтаксис функции GetNumber1 состоит из следующих частей:

Параметр Описание
sNumPair строковое выражение, определяющее строку типа "число1 / число2" из которой функция возвращает число1.

Функция VTOFS

Преобразовывает значение заданного типа в форматированное строковое представление.


Возвращается символьная строка.

 

Синтаксис

VTOFS (strType, Value)

Синтаксис функции VTOFS состоит из следующих частей:

Параметр Описание
strType строковое выражение, определяющее тип данных для преобразования. Исходя из длины типа определяется длина форматированной строки.
Value выражение типа Variant, определяющее исходное значение, подлежащее преобразованию.

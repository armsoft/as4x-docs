Функция VTOS

Преобразовывает значение заданного типа в строковое представление.

Возвращается символьная строка.

 

Синтаксис

VTOS (strType, Value)

Синтаксис функции VTOS состоит из следующих частей:

Параметр Описание
strType строковое выражение, определяющее тип данных для преобразования.
Value выражение типа Variant, определяющее исходное значение, подлежащее преобразованию.

Функция CreateCallBackOnTimer
свойства   методы

Создает новый экземпляр объекта агента циклического вызова процедур, для исполнения скриптовой процедуры в режиме, когда по истечении заданного промежутка времени, циклически вызывается данная процедура и результат отображается в модальной окне.

Синтаксис

set sNewCallBackOnTimer = CreateCallbackOnTimer (sModName, sSubName)

Синтаксис функции CreateCallBackOnTimer состоит из следующих частей:

Параметр Описание
sNewCallBackOnTimer   переменная, ссылающаяся на новый экземпляр агента циклического вызова процедур.
sModName строковое выражение, определяющее имя модуля.
sSubName строковое выражение, определяющее имя процедуры.

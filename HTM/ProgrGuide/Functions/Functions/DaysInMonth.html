Функция DaysInMonth

Возвращает количество дней для месяца указанной даты.

Возвращает значение типа целое.

 

Синтаксис

DaysInMonth (dDate)                                                                                                     

Синтаксис функции DaysInMonth состоит из следующих частей:

Параметр Описание
dDate выражение типа дата, определяющее дату.

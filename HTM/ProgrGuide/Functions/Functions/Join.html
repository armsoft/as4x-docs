Функция Join

Возвращает строку, созданную в результате объединения элементов массива. Т.е. функция объединяет элементы массива в одну строку, вставляя между ними заданный разделитель.

 

Синтаксис

Join (SourceArray, [sDelimiter])

Синтаксис функции Join состоит из следующих частей:

Параметр Описание
SourceArray одномерный массив, содержащий элементы для объединения в строку.
sDelimiter необязательное строковое выражение, определяющее символ разделителя для отделения подстрок в возвращаемой строке. По умолчанию разделителем является символ " " пробела.

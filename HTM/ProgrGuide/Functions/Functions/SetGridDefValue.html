Метод SetGridDefValue

Задает значения по умoлчанию колонкам заданных грид-таблиц  текущего  документа.

Синтаксис

SetGridDefValue(docCurrent, dicGrid)

Синтаксис функции SetGridDefValue состоит из следующих частей:

Параметр Описание

docCurrent

Обязательная  переменная, ссылающаяся на текущий документ.

dicGrid Обязательное выражение типа Dictionary, ключами которого являются имена грид-таблиц  документа.

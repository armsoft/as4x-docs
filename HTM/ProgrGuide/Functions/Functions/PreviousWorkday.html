Функция PreviousWorkDay

Возвращает ближайший предыдущий рабочий день.

Возвращает значение типа дата.

 

Синтаксис

PreviousWorkDay (dDay)                                                                                                     

Синтаксис функции PreviousWorkDay состоит из следующих частей:

Параметр Описание
dDay выражение типа дата, определяющее дату для нахождения ближайшего предыдущего рабочего дня.

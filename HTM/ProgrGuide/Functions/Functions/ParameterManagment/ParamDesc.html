Функция ParamDesc

Возвращает описание параметра.

Возвращает значение типа строки.

 

Синтаксис

ParamDesc (sParId, [sParType])

Синтаксис функции ParamDesc состоит из следующих частей:

Параметр Описание
sParid строковое выражение, определяющее код параметра из таблицы PARAMS.
sParType ссылочная переменная типа строка, возвращающая тип параметра из таблицы PARAMS.

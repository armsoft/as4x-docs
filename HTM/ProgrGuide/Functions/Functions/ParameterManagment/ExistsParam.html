Функция ExistsParam

Возвращает наличие параметра с заданным идентификатором. Если параметр существует, тогда функция возвращает также ее значение.

Возвращаемое значение логическое.

 

Синтаксис

ExistsParam (strParID, [varValue])

Синтаксис функции ExistsParam состоит из следующих частей:

Параметр Описание
strParID строковое выражение, определяющее идентификатор параметра из таблицы PARAMS.
varValue ссылочная переменная типа Variant, возвращающая значение параметра, если параметр с заданным идентификатором существует. В противном случае оно не имеет смысла.

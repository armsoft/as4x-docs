Функция GetParam

Возвращает значение параметра непосредственно из базы. Значение перманентных параметров в отличие от свойства Param, приводятся непосредственно из базы.

Возвращает значение типа Variant.

 

Синтаксис

GetParam (sParid, nSUID)

Синтаксис функции GetParam состоит из следующих частей:

Параметр Описание
sParid строковое выражение, определяющее идентификатор параметра. При задании несуществующего идентификатора генерируется ошибка.
nSUID строковое выражение, определяющее код пользователя, для которого приводится значение параметра. Для определения значений пользователе-независимых либо   пользователе-зависимых параметров по умолчанию задается значение 0.

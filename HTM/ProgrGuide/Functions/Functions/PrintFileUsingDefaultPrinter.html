Метод PrintFileUsingDefaultPrinter

Выводит файл на принтер.

 

Синтаксис

PrintFileUsingDefaultPrinter(fileName, showPrintFile )

Синтаксис функции PrintFileUsingDefaultPrinter состоит из следующих частей:

Параметр Описание
fileName

Обязательное строковое выражение, определяющее имя файла, выводимого на принтер.

showPrintFile Обязательное логическое выражение, определяющее показ отпринтованного файла.  По умолчанию принимает значение False.

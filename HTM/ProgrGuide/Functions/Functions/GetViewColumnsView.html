Функция GetViewColumnsView

Возвращает динамический вид просмотра без заданной кололнки.

 

Синтаксис

GetViewColumnsView (viewName, excludeColumns)

Синтаксис метода GetViewColumnsView состоит из следующих частей:

Параметр Описание
viewName обязательное строковое выражение, определяющее идентификатор вида просмотра, из которого исключается заданная колонка.
excludeColumns строковое выражение, определяющее исключаемую колонку.

Метод ExecuteUserReport

Выпоняет пользовательский отчет.

 

Синтаксис

ExecuteUserReport(sReportName,dicParamValues)

Синтаксис метода ExecuteUserReport состоит из следующих частей:

Параметр Описание
sReportName выражение, определяющее пользовательского отчета.
dicParamValues обьект типа Dictionary, содержащий значения параметров для подстановки в пользовательский отчет. Ключем в dicParamValues является имя параметра.

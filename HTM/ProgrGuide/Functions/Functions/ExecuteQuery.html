Функция ExecuteQuery

Выполняет SQL-запрос и возвращает результат выполнения запроса при заданном параметре.

Синтаксис

Set rs = Util.ExecuteQuery(sSQL, returnRS, [RSType], [RSLockType], [RSOption], [QueryTimeout])

Синтаксис функции ExecuteQuery состоит из следующих частей:

Параметр Описание
rs переменная типа rdoResultset.
sSQL строковое выражение, определяющее текст запроса.
returnRS логическое выражение, определяющее признак возвращения результата запроса.
RSType необязательное числовое выражение, определяющее тип курсора возвращаемого результата. По умолчанию принимает значение ASOpenForwardOnly.
RSLockType необязательное числовое выражение, определяющее тип блокировки данных, используемых при выполнении запроса. По умолчанию принимает значение ASConcurReadOnly.
RSOption необязательное числовое выражение, определяющее дополнительные параметры выполнения SQL-запроса. По умолчанию принимает значение ASExecDirect.
QueryTimeout необязательное числовое выражение, определяющее максимальное время выполнения запроса, по истечению которого выдается ошибка. Измеряется в секундах. По умолчанию принимает значение 30 секунд.

Свойство DeferredTrans

Возвращает или устанавливает признак отложенных транзакций. Если значение свойства установлено True, тогда проводки являются отложенными, т.е. они после применения StoreFact собираются в коллекцию, а потом только  запоминаются. Если значение свойства установлено False, то проводка с помощью метода StoreFact сразу сохраняется.

Свойство для чтения и записи.

 

Синтаксис

DeferredTrans[=bValue]

Синтаксис свойства DeferredTrans состоит из следующих частей:

Параметр Описание
bValue логическое выражение, определяющее признак отложенной транзакции. По умолчанию значение свойства принимается равным True.

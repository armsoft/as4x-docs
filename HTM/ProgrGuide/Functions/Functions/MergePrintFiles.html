Метод MergePrintFiles
 

Объединяет два файла печати в один.

Синтаксис

MergePrintFiles sTargetFile, sSourceFile, sTemplateType, bInsertBreak

Синтаксис функции MergePrintFiles состоит из следующих частей:

Параметр Описание
sTargetFile строковое выражение, определяющее путь целевого файла к которому будет добавлен исходный файл
sSourceFile строковое выражение, определяющее путь исходного файла который будет добавлен в целевой
sTemplateType строковое выражение, определяющее тип шаблонов печати
bInsertBreak логическое выражение, определяющее признак добавления break-а между содежимимы двух файлов

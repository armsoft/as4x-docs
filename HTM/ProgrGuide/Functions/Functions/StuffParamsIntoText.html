Функция StuffParamsIntoText

Вставляет в исходную строку вместо символов %n1,%n2,.. значения параметров.

 

Синтаксис

StuffParamsIntoText(sourceString, param1, [param2,..])

Синтаксис функции StuffParamsIntoText состоит из следующих частей:

Параметр Описание
sourceString исходная строка.
param1 строка которая будет подставлена вместо %n1 в исходную строку.
param2 строка которая будет подставлена вместо %n2 в исходную строку.

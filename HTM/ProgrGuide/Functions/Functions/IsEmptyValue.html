Функция IsEmptyValue

Возвращает значение True, если значение vValue совпадает с его системным значением по умолчанию.

 

Синтаксис

IsEmptyValue(vValue, [sDataType])

Синтаксис функции IsEmptyValue состоит из следующих частей:

Параметр Описание
vValue обязательное выражение типа вариант, значение которого проверяется.
sDataType необязательное строковое выражение, определяющее тип данных для значения vValue.

 

Примечание

Если sDataType не задан, то функция воспринимает vValue как тип VB.

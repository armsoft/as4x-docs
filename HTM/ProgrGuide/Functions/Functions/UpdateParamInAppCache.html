Метод UpdateParamInAppCache

Обновляет задаваемый параметр в кэш шлюзе.

Синтаксис

UpdateParamInAppCache(paramID)

Синтаксис функции UpdateParamInAppCache состоит из следующих частей:

Параметр Описание
paramID

Обязательное строковое выражение, определяющее идентификатор задаваемого параметра.

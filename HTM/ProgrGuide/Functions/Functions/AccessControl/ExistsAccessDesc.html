Функция ExistsAccessDesc

Возвращает признак существования описателя доступа.

Возвращаемое значение логическое.

 

Синтаксис

Util.ExistsAccessDesc (nISN, btAccessID)

Синтаксис функции ExistsAccessDesc состоит из следующих частей:

Параметр Описание
nISN длинное целое, определяющее ISN документа для которого проверяется наличие описателя доступа.
btAccessID байтовое выражение, определяющее код доступа из описания документа.

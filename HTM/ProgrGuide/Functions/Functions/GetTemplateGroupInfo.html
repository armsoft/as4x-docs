Функция GetTemplateGroupInfo

Возврашает группу шаблона и группу данных, передаваемых шаблону.

Возврашаемое значение типа Dictionary.

Синтаксис

GetTemplateGroupInfo (TemplateName, TemplateType, [ByRef TemplateGroup])

Синтаксис функции GetTemplateGroupInfo состоит из следующих частей:

Параметр Описание
TemplateName Обязательное строковое выражение, определяющее имя шаблона TEMPLATE
TemplateType Обязательное численное выражение, определяющее тип шаблона
TemplateGroup Необязательная ссылочная переменная строкового типа, определяющая группу шаблона

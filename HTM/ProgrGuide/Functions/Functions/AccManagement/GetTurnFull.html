--- layout: page title: "GetTurnFull ֆունկցիա" --- ## Функция GetTurnFull Возвращает значения оборотов и остатков объекта учета. Синтаксис GetTurnFull (sAccCode, nISN, dStartDate, dEndDate, DbTurn, DbTurnAMD, _
                CrTurn, CrTurnAMD, Remd, RemdAMD, [StartRemd], [StartRemdAMD]) Синтаксис процедуры GetTurnFull состоит из следующих частей: | Параметр | Описание | |--|--| | sAccCode | строковое выражение, определяющее [код учета](../../../Defs/Accounting.html) | | nISN | числовое выражение Long, определяющее ISN объекта учета | | dStartDate | выражение типа даты, определяющее начальную дату расчета оборота | | dEndDate | выражение типа даты, определяющее конечную дату расчета оборота | | DbTurn | ссылочная переменная, возвращающая дебетовый оборот в иностранной валюте | | DbTurnAMD | ссылочная переменная, возвращающая дебетовый оборот в основной валюте | | CrTurn | ссылочная переменная, возвращающая кредитовый оборот в иностранной валюте | | CrTurnAMD | ссылочная переменная, возвращающая кредитовый оборот в основной валюте | | Remd | ссылочная переменная, возвращающая конечный остаток объекта в иностранной валюте. Остаток приводится на конец дня. | | RemdAMD | ссылочная переменная, возвращающая конечный остаток объекта в национальной валюте. Остаток приводится на конец дня. | | StartRemd | необязательный параметр, ссылочная переменная, возвращающая начальный остаток объекта в иностранной валюте. Остаток приводится на начало дня. | | StartRemdAMD | необязательный параметр, ссылочная переменная, возвращающая начальный остаток объекта в национальной валюте. Остаток приводится на начало дня. | Примечание [См. также](GetTurn.html)

Dim DbTurn, DbTurnAMD
Dim CrTurn, CrTurnAMD
Dim Remd, RemdAMD
Dim StartRemd, StartRemdAMD

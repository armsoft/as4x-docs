## GetRemHI2 ֆունկցիա

Վերադարձնում է օբյեկտի մնացորդները ըստ կուտակող օբյեկտի դրամային արտարժույթով նշված ամսաթվի կամ ընթացիկ մնացորդի վերջում։ 
Возвращает остатки объекта по заданному аккумулирующему объекту в иностранной и основной валютах на конец указанной даты либо текущий остаток.

## Շարահյուսություն

```vb
GetRemHI2 (sAccounting, nISN, nGLISN, [dRemDate], [AMDRem])
```
Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| sAccounting | [Հաշվառման նկարագրության կոդը](../../../Defs/Accounting.html): строковое выражение, определяющее [код учета](../../../Defs/Accounting.html) из его описания. |
| nISN | Հաշվառում օբյեկտի ներքին նույնականացման համար, որի համար բերվում են մնացորդներ։ численное выражение типа Long, определяющее внутрисистемный идентификационный код ISN объекта учета, для которого приводятся остатки |
| nGLISN | Կուտակող օբյեկտի ներքին նույնականացման համար։ числовое выражение Long, определяющее ISN аккумулирующего объекта |
| dRemDate | Մնացորդը բերելու ամսաթիվը։ Եթե պարամետրը առկա չի, ապա բերվում է ընթացիկ մնացորդը։ необязательное выражение типа даты, определяющее дату приведения остатка. Если параметр опущен, то приводится текущий остаток. |
| AMDRem | Պարամետր, որը ստանում է մնացորդը դրամային արտարժույթով։ необязательный параметр получения остатка в основной валюте. |

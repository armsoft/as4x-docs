Функция Between

Определяет находится ли значение в рамках указанного диапазона.

Синтаксис

Between(Value, MinValue, MaxValue)

Синтаксис функции Between состоит из следующих частей:

Параметр Описание
Value Выражение значение которого проверяется.
MinValue Нижний предел проверяемого диапазона.
MaxValue Верхний предел проверяемого диапазона.

Параметры SQL-запроса источника данных

 

Синтаксис

Param {
                  Name = sParName;
                  Description = sParamDescription;
                  Type = sParamType;
                  UserReportValue = sParamValue;
             };

  Значение Описание

sParName

необязательное строковое выражение, определяющее идентификатор параметра

sParamDescription

строковое выражение, определяющее описание параметра

sParamType

строковое выражение, определяющее внутренний тип параметра, подставляемого в SQL запрос

sParamValue

необязательное строковое выражение, определяющее значение параметра в генераторе отчетов. Указанное значение не может быть изменено пользователем при выполнеии отчета.

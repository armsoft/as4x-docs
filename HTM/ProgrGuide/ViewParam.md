# Դիտելու ձևի պարամետրերի նկարագրություն


## Շարահյուսություն

``` as4x
ParamValue  {
                Name = sName;
                Value = sValue;
                };
```

| Արժեք | Նկարագրություն |
|--|--|
| sName | Դիտելու ձևի պարամետրերի ներքին անուն։ строковое выражение, определяющее идентификатор параметра в источнике данных.|
| sValue | Դիտելու ձևի պարամետրերի արժեք։ строковое выражение, определяющее значение параметра.|


## Նկատառումներ

Իմաստ ունի պարամետրերին արժեքներ տալ եթե նրանք տարբերվում են լռության արժեքներից։

Имеет смысл задавать значения параметров если они отличаются от значений по умолчанию.

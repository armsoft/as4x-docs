Константы контекста события Functions документа

Т.к. эти константы объявлены в системном ядре, то нет необходимости в их дополнительном объявлении. Их можно использовать везде в скриптах, взамен их значений.

Константа Значение Описание
ASFCView 0 операции над документом регистрируются из вида просмотра
ASFCTree 1 операции над документом регистрируются из дерева
ASFCReportView 2 операции над документом регистрируются из отчета
ASFCMessengerView 3 операции над документом регистрируются из вестника


Эти константы могут быть также использованы в качестве параметров события Functions документа.

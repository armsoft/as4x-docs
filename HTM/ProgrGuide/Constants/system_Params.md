# Համակարգային պարամետրեր

|--|--|--|
| OPEREND| DATE | Գործառնական ժամանակաշրջանի վերջ։ |
| OPERSTART| DATE | Գործառնական ժամանակաշրջանի սկիզբ։ |

| EMANAGERNAME| C(32) | Կազմակերպության ղեկավարի Ա.Ա.Հ-ն օտար լեզվով։ |

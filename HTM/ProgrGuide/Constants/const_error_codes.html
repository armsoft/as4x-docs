Константы кодов ошибок

Т.к. эти константы объявлены в системном ядре, то нет необходимости в их дополнительном объявлении. Их можно использовать везде в скриптах, взамен их значений.

Константа Значение Описание
gintUserErrors vbObjectError + 2 код ошибки при сгенерировании которой программа останавливается и выдает соответствующее сообщение пользователю.
gintUserBreakError vbObjectError + 3 код ошибки при сгенерировании которой программа останавливается.

Константы указателя мыши

Т.к. эти константы объявлены в системном ядре, то нет необходимости в их дополнительном объявлении. Их можно использовать везде в скриптах, взамен их значений.

Константа Значение Описание
vbHourGlass 11 указатель мыши принимает вид песочных часов
vbDefault 0 указатель мыши принимает обычный вид стрелки


Эти константы могут быть использованы в качестве значения свойства MousePointer, для определения вида указателя мыши.

Константы индикатора

Т.к. эти константы объявлены в системном ядре, то нет необходимости в их дополнительном объявлении. Их можно использовать везде в скриптах, взамен их значений.

Константа Значение Описание
ASIndicateHourglass 0 появляются только песочные часы
ASIndicateProgressBar 1 появляется индикатор продвижения выполнения запроса
ASIndicateNone 2 отсутствуют и песочные часы и индикатор


Эти константы могут быть также использованы в качестве параметров метода OpenCursor источника данных, для определения вида появляющегося индикатора продвижения процесса.

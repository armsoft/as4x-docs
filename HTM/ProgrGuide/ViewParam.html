Параметры вида просмотра

 

Синтаксис

ParamValue  {
                            Name = sName;
                            Value = sValue;
                         };

  
  Значение Описание

sName

строковое выражение, определяющее идентификатор параметра в источнике данных.

sValue

строковое выражение, определяющее значение параметра.

Примечание

Имеет смысл задавать значения параметров если они отличаются от значений по умолчанию.

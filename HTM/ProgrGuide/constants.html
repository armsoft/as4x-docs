Системные константы и параметры

Некоторые полезные константы, которые можно использовать в скриптах, встроены в ядро системы. Константы обеспечивают удобное использование особых значений без запоминания самих значений. Использование констант делает  программы более сопровождаемыми при изменении значений этих констант. Т.к. эти константы уже объявлены на уровне ядра, то нет нужды объявлять их в скриптах. Просто вместо значений надо использовать имена констант.

Здесь приведены различные группы констант и их краткое описание:

Константы состояния документа
Определяет состояния документа, в котором он может находиться.

Константы индикатора
Определяет константы вывода индикатора продвижения процесса.

Константы типов блокировок
Определяет константы существующих видов блокировок.

Константы типов курсоров
Определяет константы имеющихся типов курсоров для источников данных.

Константы типов выполнения SQL-запроса
Определяет константы, определяющие типы выполнения SQL-запроса.

Константы указателя мыши
Определяет константы, определяющие виды указателя мыши.

Константы контекста события Functions документа
Определяет константы, определяющие контекст события Functions документа.

Константы кодов ошибок
Определяет константы, определяющие коды ошибок при генерировании которых программа приостанавливается.

Системные параметры
Приводится список и поясняется назначение общесистемных параметров.

Константы типов пользовательских функций в контекстном меню вызовов.
Приводится список типов пользовательских функций в контекстном меню вызовов.

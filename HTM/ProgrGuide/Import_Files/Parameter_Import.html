Описание текстового импортного файла ситемного параметра

См. также Пример Применяется к

Синтаксис

#AS3XX# EXPORT-IMPORT DATA FILE

PARAMETER {
PARAMID:paramId1    
TYPE:paramType1
VALUETYPE:paramValueType1
VALUE:paramValue1
DESCRIPTION:paramDescr1
EDESCRIPTION:paramEDescr1
PERMANENT:parampermanent1
GROUP:paramGroup1
UPDATEACCESS:paramApdateAccess1
MEMO:paramMemo1
}

PARAMETER {
PARAMID:paramId2    
TYPE:paramType2
VALUETYPE:paramValueType2
VALUE:paramValue2
DESCRIPTION:paramDescr2
EDESCRIPTION:paramEDescr2
PERMANENT:paramPermanent2
GROUP:paramGroup2
UPDATEACCESS:paramApdateAccess2
MEMO:paramMemo2
}

. . .
  



 Значение

 Описание

paramId

обязательное строковое выражение, определяющее идентификатор параметра, максимальная длина которого не больше 20 -и символов.

paramType

обязательное строковое выражение, определяющее тип параметра. Принимает значения 0 или 1. При значении 1 дается возможность редактирования параметра на уровне пользователя.

paramValueType

обязательное строковое выражение, определяющее внутренний тип значения параметра.

paramValue

необязательное строковое выражение, определяющее значение парамтера.

paramDescr

необязательное строковое выражение, определяющее описание параметра.

paramEDescr

необязательное строковое выражение, определяющее описание параметра на иностранном языке.

paramPermanent

обязательное строковое выражение, определяющее постоянство параметра. Принимает значения: 0 - при входе в систему или обращении к параметру, его значение не сохраняется в ОЗУ.
                  1 - значение параметра сохранаяется и при входе в систему и обращении к неу.
                  2 - при входе в систему значение параметра  не сохранаяется в ОЗУ, но после чтении или обращении к нему  остается там.

paramGroupspan>

необязательное строковое выражение, определяющее группу параметра.

paramApparamApdateAccess

обязательное строковое выражение, определяющее разрешимость редактирования параметра. Принимает значения 0, 1 или 2. При значении 0 редактирование параметра не разрешается, при значении 1 разрешается только администратору, а при значении 2 всем поьзователям системы.

paramMemo

необязательное строковое выражение, определяющее подробное описание параметра.

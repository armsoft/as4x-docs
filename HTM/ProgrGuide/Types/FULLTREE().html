Тип данных FullTree( )

Ссылка на справочник типа дерева. Из справочника данного типа, в отличие от типа Tree, могут быть выбраны листья из  любого уровня.

 

Синтаксис

FullTree(sTreeID)

Синтаксис типа FullTree состоит из следующих частей:

Параметр Описание
sTreeID символьное выражение, определяющее идентификатор дерева-справочника. Максимально допустимая длина идентификатора дерева 8 символов.

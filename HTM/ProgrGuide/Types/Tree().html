Тип данных Tree( )

Ссылка на справочник типа дерева. Из справочника данного типа, в отличие от типа FullTree, могут быть выбраны только листья последнего уровня.

 

Синтаксис

Tree (sTreeId)

Синтаксис типа Tree состоит из следующих частей:

Параметр Описание
sTreeId символьное выражение, определяющее идентификатор дерева-справочника. Максимально допустимая длина идентификатора дерева 8 символов.

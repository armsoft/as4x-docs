# C() տվյալների տիպ

Տողային տվյալների տիպ։


## Շարահյուսություն

```
C(nLen) 
```

Բաղադրիչներն են՝


| Պարամետր | Նկարագրություն |
|--|--|
| nLen | Տողային արժեքի մեջ առավելագույն Նիշերի քանակ։ Առավելագույնը՝ 512։ |




[См. также](../types.html)


## Օրինակ

C(50) - Սիմվոլային տող առավելագույնը 50 երկարությամբ։

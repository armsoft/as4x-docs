Тип данных DateLong

Тип даты. Принимает значения в интервале от 01/01/0100 до 31/12/9999. Значения данного типа имеют формат dd/mm/yyyy.

 

Синтаксис

DateLong

См. также

 

Примечание

Данный тип не используется в системе для указания типов значений колонок и передаваемых параметров в источниках данных.

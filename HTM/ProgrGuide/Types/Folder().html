Тип данных Folder( )

Ссылка на элементы папки.

 

Синтаксис

Folder(sFolderId, nKeyLen)

Синтаксис типа Folder состоит из следующих частей:

Параметр Описание
sFolderId строковое выражение, определяющее идентификатор папки. Максимально допустимая длина идентификатора папки 20 символов.
nKeyLen числовое выражение, определяющее длину ключа папки.

Пример использования ссылки на папку

FOLDER(ACC,11) - ссылка на папку ACC с длиной ключа 11.
